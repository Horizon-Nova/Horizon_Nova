Function(parentNode)zTreeObj.removeChildNodes

概述[ 依赖 jquery.ztree.core 扩展 js ]

清空某父節點的子節點。

1、清空子節點後，父節點会自動變為叶子節點，如需要父節點保持父節點狀態，請設置 setting.data.keep.parent 属性。

2、請勿用此方法清空根節點，如果需要清空根節點，直接初始化 zTree，並且設置初始節點為 null 即可。

3、此方法不会触發任何事件回調函數。

請通過 zTree 對象執行此方法。

Function 參數說明

parentNodeJSON

需要清空子節點的父節點資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

返回值Array(JSON)

將該父節點的子節點資料返回，如果不存在则返回 null

function 举例

1. 清空選中的第一個節點的子節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes && nodes.length>0) {
	treeObj.removeChildNodes(nodes[0]);
}
