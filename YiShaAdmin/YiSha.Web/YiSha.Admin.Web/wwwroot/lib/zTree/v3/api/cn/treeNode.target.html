StringtreeNode.target

概述[ 依赖 jquery.ztree.core 核心 js ]

設置點击節點後在何處打開 url。[treeNode.url 存在時有效]

默認值：无

String 格式說明

同超連結 target 属性: "_blank", "_self" 或 其他指定窗口名稱

省略此属性，则默認為 "_blank"

treeNode 举例

1. 設置點击某節點時，弹出新頁面

var nodes = [
	{ "id":1, "name":"test1", "url":"http://myTest.com", "target":"_blank"},
	......
]
