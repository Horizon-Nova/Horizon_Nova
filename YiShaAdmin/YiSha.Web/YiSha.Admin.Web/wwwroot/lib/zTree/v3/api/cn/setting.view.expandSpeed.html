String / Numbersetting.view.expandSpeed

概述[ 依赖 jquery.ztree.core 核心 js ]

zTree 節點展開、折叠時的動画速度，設置方法同 JQuery 動画效果中 speed 參數。

IE6 下会自動關閉動画效果，以保證 zTree 的操作速度

默認值："fast"

String 格式說明

三種預定速度之一的字符串("slow", "normal", or "fast")

設置為 "" 時，不顯示動画效果

Number 格式說明

表示動画時长的毫秒數值 (如：1000)

setting 举例

1. 設置為慢速顯示動画效果

var setting = {
	view: {
		expandSpeed: "slow"
	}
};
