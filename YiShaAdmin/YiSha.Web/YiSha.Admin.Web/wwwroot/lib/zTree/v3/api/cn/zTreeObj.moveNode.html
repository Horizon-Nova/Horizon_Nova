Function(targetNode, treeNode, moveType, isSilent)zTreeObj.moveNode

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

移動節點。

請通過 zTree 對象執行此方法。

Function 參數說明

targetNodeJSON

要移動到的目標節點 JSON 資料

如果移動成為根節點，請設置 targetNode 為 null 即可

請務必保證此節點資料對象 是 zTree 內部的資料對象

treeNodeJSON

需要被移動的節點資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

moveTypeString

指定移動到目標節點的相對位置

"inner"：成為子節點，"prev"：成為同級前一個節點，"next"：成為同級後一個節點

isSilentBoolean

設定移動節點後是否自動展開父節點。

isSilent = true 時，不展開父節點，其他值或缺省狀態都自動展開。

返回值JSON

返回值是最終被移動的節點資料，正常情况下與 treeNode 參數完全相同

如果 返回值 為 null，說明 移動失敗，主要原因有：
 1、targetNode 是 treeNode 父節點，且 moveType = "inner"
 2、targetNode 是 treeNode 子孙節點

function 举例

1. 將根節點中第二個節點 移動成為 第一個節點的子節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.moveNode(nodes[0], nodes[1], "inner");

2. 將根節點中第二個節點 移動成為 第一個節點的前一個節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.moveNode(nodes[0], nodes[1], "prev");
