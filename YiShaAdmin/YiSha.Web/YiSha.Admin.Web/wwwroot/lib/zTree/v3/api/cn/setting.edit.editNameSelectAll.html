Booleansetting.edit.editNameSelectAll

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

節點編輯名稱 input 初次顯示時,設置 txt 內容是否為全選狀態。 [setting.edit.enable = true 時生效]

默認值: false

Boolean 格式說明

true 表示 全選狀態

false 表示 不是全選狀態，光標默認在最後

setting 举例

1. 設置節點編輯名稱 input 初次顯示時，txt內容為全選狀態

var setting = {
	edit: {
		enable: true,
		editNameSelectAll: true
	}
};
