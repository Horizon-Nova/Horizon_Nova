Boolean / Function(treeId, treeNode)setting.edit.showRemoveBtn

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

設置是否顯示删除按鈕。[setting.edit.enable = true 時生效]

當點击某節點的删除按鈕時：

1、首先触發 setting.callback.beforeRemove 回調函數，使用者可判定是否進行删除操作。

2、如果未設置 beforeRemove 或 beforeRemove 返回 true，则删除節點並触發 setting.callback.onRemove 回調函數。

默認值：true

Boolean 格式說明

true / false 分別表示 顯示 / 隱藏 删除按鈕

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

需要設置是否顯示删除按鈕的節點 JSON 資料對象

返回值Boolean

返回值同 Boolean 格式的資料

setting & function 举例

1. 不顯示删除按鈕

var setting = {
	edit: {
		enable: true,
		showRemoveBtn: false
	}
};
......

2. 設置所有的父節點不顯示删除按鈕

function setRemoveBtn(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		showRemoveBtn: setRemoveBtn
	}
};
......
