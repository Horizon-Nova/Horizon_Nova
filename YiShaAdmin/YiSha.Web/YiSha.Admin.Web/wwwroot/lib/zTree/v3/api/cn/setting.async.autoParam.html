Array(String)setting.async.autoParam

概述[ 依赖 jquery.ztree.core 核心 js ]

異步加載時需要自動提交父節點属性的參數。[setting.async.enable = true 時生效]

默認值：[ ]

Array(String) 格式說明

1、將需要作為參數提交的属性名稱，制作成 Array 即可，例如：["id", "name"]

2、可以設置提交時的參數名稱，例如 server 只接受 zId : ["id=zId"]

setting 举例

1. 設置 id 属性為自動提交的參數

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id"]
	}
};
假設 異步加載 父節點(node = {id:1, name:"test"}) 的子節點時，將提交參數 id=1
......

2. 設置 id 属性作為 zId 成為自動提交的參數

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id=zId"]
	}
};
假設 對父節點 node = {id:1, name:"test"}，進行異步加載時，將提交參數 zId=1
......
