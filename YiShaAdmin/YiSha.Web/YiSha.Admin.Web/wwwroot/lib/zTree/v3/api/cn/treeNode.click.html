StringtreeNode.click

概述[ 依赖 jquery.ztree.core 核心 js ]

最簡單的 click 事件操作。相當於 onclick="..." 的內容。 如果操作较复杂，請使用 onClick 事件回調函數。

由於 IE 對於 onclick 和 click事件共存時的處理與其他瀏覽器不同，所以請不要利用此參數控制是否允許跳转的操作（例如：treeNode.click = "return false;"）。如有類似需求，請不要使用 url 属性設置网址，同時利用 onClick 回調函數控制跳转。

默認值：无

String 格式說明

標准 javascript 語法， 例如：alert("test"); 等

treeNode 举例

1. 設置某節點點擊時，弹出信息框

var nodes = [
	{ "id":1, "name":"Google CN", "url":"http://g.cn", "click":"alert('test');"},
	......
]
