Booleansetting.edit.drag.isMove

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

拖拽時, 設置是否允許移動節點。[setting.edit.enable = true 時生效]

默認值：true

规则說明

1、isCopy = true; isMove = true 時，拖拽節點按下 Ctrl 或 Cmd 鍵表示 copy; 否则為 move

2、isCopy = true; isMove = false 時，所有拖拽操作都是 copy

3、isCopy = false; isMove = true 時，所有拖拽操作都是 move

4、isCopy = false; isMove = false 時，禁止拖拽操作

setting 举例

1. 設置所有拖拽操作都是 move

var setting = {
	edit: {
		enable: true,
		drag: {
			isCopy: false,
			isMove: true
		}
	}
};
