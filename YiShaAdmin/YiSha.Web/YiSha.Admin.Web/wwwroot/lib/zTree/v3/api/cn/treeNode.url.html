節點連結的目標 URL

1、編輯模式 (setting.edit.enable = true) 下此属性功能失效，如果必須使用類似功能，請利用 onClick 事件回調函數自行控制。

2、如果需要在 onClick 事件回調函數中進行跳转控制，那么請將 URL 地址保存在其他自定義的属性內，請勿使用 url
