Function(treeNode, addFlag)zTreeObj.selectNode

概述[ 依赖 jquery.ztree.core 核心 js ]

選中指定節點

v3.x 支持同時選中多個節點。

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

需要被選中的節點資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

addFlagBoolean

addFlag = true 表示追加選中，会出現多點同時被選中的情况

addFlag = false 表示單独選中，原先被選中的節點会被取消選中狀態

setting.view.selectedMulti = false 時，此參數无效，始終進行單独選中

返回值无

目前无任何返回值

function 举例

1. 單独選中根節點中第一個節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
if (nodes.length>0) {
	treeObj.selectNode(nodes[0]);
}
