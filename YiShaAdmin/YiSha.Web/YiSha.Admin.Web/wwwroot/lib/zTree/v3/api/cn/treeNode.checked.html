BooleantreeNode.checked

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

節點的 checkBox / radio 的 勾選狀態。[setting.check.enable = true & treeNode.nocheck = false 時有效]

1、如果不使用 checked 属性設置勾選狀態，請修改 setting.data.key.checked

2、建立 treeNode 資料時設置 treeNode.checked = true 可以讓節點的輸入框默認為勾選狀態

3、修改節點勾選狀態，可以使用 treeObj.checkNode / checkAllNodes / updateNode 方法，具體使用哪種請根据自己的需求而定

4、為了解决部分朋友生成 json 資料出現的兼容問題, 支持 "false","true" 字符串格式的資料

默認值：false

Boolean 格式說明

true 表示節點的輸入框被勾選

false 表示節點的輸入框未勾選

treeNode 举例

1. 初始化的資料設置 默認為勾選狀態

var nodes = [
{ "id":1, "name":"test1", checked:true },
{ "id":2, "name":"test2", checked:true }
]

2. 獲取第一個根節點的勾選狀態

var treeObj = $.fn.zTree.getZTreeObj("tree");
var checked = treeObj.getNodes()[0].checked;
