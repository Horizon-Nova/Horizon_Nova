Function()zTreeObj.getNodes

概述[ 依赖 jquery.ztree.core 核心 js ]

獲取 zTree 的全部節點資料

請通過 zTree 對象執行此方法。

Function 參數說明

返回值Array(JSON)

全部節點資料

1、Array 僅僅是根節點的集合（默認情况子節點都處於 children 属性下）；

2、如需遍历全部節點需要利用递归，或利用 transformToArray 方法 將資料變成簡單的 Array 集合

3、對於異步加載模式下，尚未加載的子節點是无法通過此方法獲取的。

function 举例

1. 獲取全部節點資料

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
