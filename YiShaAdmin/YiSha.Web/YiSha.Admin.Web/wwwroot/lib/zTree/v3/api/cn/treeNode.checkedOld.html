BooleantreeNode.checkedOld

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

節點的 checkBox / radio 在初始化時的 勾選狀態。[setting.check.enable = true & treeNode.nocheck = false 時有效]

1、zTree 初始化節點資料時会對此属性進行赋值，因此請勿對此属性初始化

2、如需配合 zTreeObj.getChangeCheckedNodes 方法實現特殊功能，可以根据需求在使用中自行修改 checkedOld 資料

默認值：checked的初始化值

Boolean 格式說明

true 表示節點初始化時 輸入框被勾選

false 表示節點初始化時 輸入框未勾選

treeNode 举例

1. 獲取第一個根節點的初始勾選狀態

var treeObj = $.fn.zTree.getZTreeObj("tree");
var checkedOld = treeObj.getNodes()[0].checkedOld;
