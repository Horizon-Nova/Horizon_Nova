Booleansetting.edit.drag.autoExpandTrigger

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

拖拽時父節點自動展開是否触發 onExpand 事件回調函數。[setting.edit.enable = true 時生效]

默認值：false

Boolean 格式說明

true / false 分別表示 触發 / 不触發 onExpand 事件回調函數。

setting 举例

1. 設置拖拽時父節點自動展開触發 onExpand 事件回調函數

var setting = {
	edit: {
		enable: true,
		drag: {
			autoExpandTrigger: true
		}
	}
};
