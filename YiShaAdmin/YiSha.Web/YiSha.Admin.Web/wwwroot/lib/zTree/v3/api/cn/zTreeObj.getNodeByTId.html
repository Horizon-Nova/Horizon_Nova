Function(tId)zTreeObj.getNodeByTId

概述[ 依赖 jquery.ztree.core 核心 js ]

根据 zTree 的唯一標識 tId 快速獲取節點 JSON 資料對象

通過內部的 cache 獲取，不需要遍历節點。

請通過 zTree 對象執行此方法。

Function 參數說明

tIdString

節點在 zTree 內的唯一標識 tId

返回值JSON

tId 對應的節點 JSON 資料對象

如无結果，返回 null

function 举例

1. 獲取 tId = "tree_10" 的節點資料

var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodeByTId("tree_10");
