Function(event, treeId, treeNodes, targetNode, moveType, isCopy)setting.callback.onDrop

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用於捕獲節點拖拽操作結束的事件回調函數

如果設置了 setting.callback.beforeDrop 方法，且返回 false，將无法触發 onDrop 事件回調函數。

默認值：null

Function 參數說明

eventjs event 對象

標准的 js event 對象

treeIdString

目標節點 targetNode 所在 zTree 的 treeId，便於使用者操控

treeNodesArray(JSON)

被拖拽的節點 JSON 資料集合

如果拖拽操作為 移動，treeNodes 是當前被拖拽節點的資料集合。

如果拖拽操作為 复制，treeNodes 是复制後 clone 得到的新節點資料。

targetNodeJSON

成為 treeNodes 拖拽結束的目標節點 JSON 資料對象。

如果拖拽成為根節點，则 targetNode = null

moveTypeString

指定移動到目標節點的相對位置

"inner"：成為子節點，"prev"：成為同級前一個節點，"next"：成為同級後一個節點

如果 moveType = null，表明拖拽无效

isCopyBoolean

拖拽節點操作是 复制 或 移動

true：复制；false：移動

setting & function 举例

1. 每次拖拽操作結束後， 弹出該被拖拽節點的個數以及目標節點的 tId、name 的信息

function zTreeOnDrop(event, treeId, treeNodes, targetNode, moveType) {
    alert(treeNodes.length + "," + (targetNode ? (targetNode.tId + ", " + targetNode.name) : "isRoot" ));
};
var setting = {
	callback: {
		onDrop: zTreeOnDrop
	}
};
