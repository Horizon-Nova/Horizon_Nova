Boolean / Function(treeId, treeNode)setting.edit.showRenameBtn

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

設置是否顯示編輯名稱按鈕。[setting.edit.enable = true 時生效]

當點击某節點的編輯名稱按鈕時：

1、進入節點編輯名稱狀態。

2、編輯名稱完畢（Input 失去焦點 或 按下 Enter 鍵），会触發 setting.callback.beforeRename 回調函數，使用者可根据自己的规则判定是否允許修改名稱。

3、如果 beforeRename 返回 false，则继续保持編輯名稱狀態，直到名稱符合规则位置 （按下 ESC 鍵可取消編輯名稱狀態，恢复原名稱）。

4、如果未設置 beforeRename 或 beforeRename 返回 true，则結束節點編輯名稱狀態，更新節點名稱，並触發 setting.callback.onRename 回調函數。

默認值：true

Boolean 格式說明

true / false 分別表示 顯示 / 隱藏 編輯名稱按鈕

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

需要設置是否顯示編輯名稱按鈕的節點 JSON 資料對象

返回值Boolean

返回值同 Boolean 格式的資料

setting & function 举例

1. 不顯示編輯名稱按鈕

var setting = {
	edit: {
		enable: true,
		showRenameBtn: false
	}
};
......

2. 設置所有的父節點不顯示編輯名稱按鈕

function setRenameBtn(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		showRenameBtn: setRenameBtn
	}
};
......
