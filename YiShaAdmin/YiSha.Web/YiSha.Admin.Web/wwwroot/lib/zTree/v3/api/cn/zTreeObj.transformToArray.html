Function(treeNodes)zTreeObj.transformToArray

概述[ 依赖 jquery.ztree.core 核心 js ]

將 zTree 使用的標准 JSON 嵌套格式的資料转換為簡單 Array 格式。(免去使用者自行編寫递归遍历全部節點的麻烦)

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodesArray(JSON) / JSON

需要被转換的 zTree 節點資料對象集合 或 某個單独節點的資料對象

返回值Array(JSON)

转換後的簡單 Array 資料格式

function 举例

1. 將 zTree 節點資料转換為簡單 Array 格式

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.transformToArray(treeObj.getNodes());
