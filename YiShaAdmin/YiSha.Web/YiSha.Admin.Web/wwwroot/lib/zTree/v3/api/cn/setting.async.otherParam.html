Array(String) / JSONsetting.async.otherParam

概述[ 依赖 jquery.ztree.core 核心 js ]

Ajax 請求提交的静態參數鍵值對。[setting.async.enable = true 時生效]

默認值：[ ]

Array(String) 格式說明

可以為空[ ]，如果有 key，则必須存在 value。 例如：[key, value]

JSON 格式說明

直接用 JSON 格式制作鍵值對，例如：{ key1:value1, key2:value2 }

setting 举例

1. 設置 Array(String) 格式的參數

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		otherParam: ["id", "1", "name", "test"]
	}
};
進行異步加載時，將提交參數 id=1&name=test

2. 設置 JSON 格式的參數

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		otherParam: { "id":"1", "name":"test"}
	}
};
進行異步加載時，將提交參數 id=1&name=test
