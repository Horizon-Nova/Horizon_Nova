BooleantreeNode.open

概述[ 依赖 jquery.ztree.core 核心 js ]

記錄 treeNode 節點的 展開 / 折叠 狀態。

1、初始化節點資料時，如果設定 treeNode.open = true，则会直接展開此節點

2、叶子節點 treeNode.open = false

3、為了解决部分朋友生成 json 資料出現的兼容問題, 支持 "false","true" 字符串格式的資料

4、非異步加載模式下，无子節點的父節點設置 open=true 後，可顯示為展開狀態，但異步加載模式下不会生效。（v3.5.15+）

默認值：false

Boolean 格式說明

true 表示節點為 展開 狀態

false 表示節點為 折叠 狀態

treeNode 举例

1. 查看當前被選中的節點的 展開 / 折叠 狀態

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isOpen = sNodes[0].open;
}
