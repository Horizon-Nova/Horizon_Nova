Function(treeNode)zTreeObj.cancelSelectedNode

概述[ 依赖 jquery.ztree.core 核心 js ]

取消節點的選中狀態。

v3.x 支持多點同時選中，因此取消選中狀態可以全部取消，也可以單独取消某個節點的選中狀態。

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

需要取消選中狀態的節點。

請務必保證此節點資料對象 是 zTree 內部的資料對象

如果省略此參數，则將取消全部被選中節點的選中狀態。

返回值无

目前无任何返回值

function 举例

1. 取消當前所有被選中節點的選中狀態

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelSelectedNode();

2. 取消當前第一個被選中節點的選中狀態

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) { 
	treeObj.cancelSelectedNode(nodes[0]);
}
