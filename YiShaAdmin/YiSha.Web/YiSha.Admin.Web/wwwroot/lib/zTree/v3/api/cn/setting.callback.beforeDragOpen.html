Function(treeId, treeNode)setting.callback.beforeDragOpen

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用於捕獲拖拽節點移動到折叠狀態的父節點後，即將自動展開該父節點之前的事件回調函數，並且根据返回值確定是否允許自動展開操作

默認值：null

Function 參數說明

treeIdString

需要被展開的父節點 treeNode 所在 zTree 的 treeId，便於使用者操控

treeNodeJSON

要被自動展開的父節點 JSON 資料對象

返回值Boolean

返回值是 true / false

如果返回 false，zTree 將无法進行自動展開操作

setting & function 举例

1. 禁止全部拖拽時的自動展開操作

function zTreeBeforeDragOpen(treeId, treeNode) {
    return false;
};
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeDragOpen: zTreeBeforeDragOpen
	}
};
