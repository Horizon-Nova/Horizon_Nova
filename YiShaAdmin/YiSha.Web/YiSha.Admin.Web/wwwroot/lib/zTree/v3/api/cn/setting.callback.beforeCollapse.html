Function(treeId, treeNode)setting.callback.beforeCollapse

概述[ 依赖 jquery.ztree.core 核心 js ]

用於捕獲父節點折叠之前的事件回調函數，並且根据返回值確定是否允許折叠操作

默認值：null

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

要折叠的父節點 JSON 資料對象

返回值Boolean

返回值是 true / false

如果返回 false，zTree 將不会折叠節點，也无法触發 onCollapse 事件回調函數

setting & function 举例

1. 禁止所有已展開的父節點折叠

function zTreeBeforeCollapse(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeCollapse: zTreeBeforeCollapse
	}
};
