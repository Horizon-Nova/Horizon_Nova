JSONsetting.check.chkboxType

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

勾選 checkbox 對於父子節點的關联關系。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" 時生效]

默認值：{ "Y": "ps", "N": "ps" }

JSON 格式說明

Y 属性定義 checkbox 被勾選後的情况；
N 属性定義 checkbox 取消勾選後的情况；
"p" 表示操作会影响父級節點；
"s" 表示操作会影响子級節點。

請注意大小寫，不要改變

setting 举例

1. checkbox 勾選操作，只影响父級節點；取消勾選操作，只影响子級節點

var setting = {
	check: {
		enable: true,
		chkStyle: "checkbox",
		chkboxType: { "Y": "p", "N": "s" }
	}
};
