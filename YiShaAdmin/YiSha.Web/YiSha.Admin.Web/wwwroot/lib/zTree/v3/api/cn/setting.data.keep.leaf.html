Booleansetting.data.keep.leaf

概述[ 依赖 jquery.ztree.core 核心 js ]

zTree 的節點叶子節點属性锁，是否始終保持 isParent = false

默認值：false

Boolean 格式說明

true / false 分別表示 锁定 / 不锁定 叶子節點属性

如果設置為 true，则所有 isParent = false 的節點，都无法添加子節點。

setting 举例

1. 需要锁定叶子節點狀態

var setting = {
	data: {
		keep: {
			leaf: true
		}
	}
};
