Function(expandFlag)zTreeObj.expandAll

概述[ 依赖 jquery.ztree.core 核心 js ]

展開 / 折叠 全部節點

此方法不会触發 beforeExpand / onExpand 和 beforeCollapse / onCollapse 事件回調函數。

請通過 zTree 對象執行此方法。

Function 參數說明

expandFlagBoolean

expandFlag = true 表示 展開 全部節點

expandFlag = false 表示 折叠 全部節點

返回值Boolean

返回值表示最終實際操作情况

true 表示 展開 全部節點

false 表示 折叠 全部節點

null 表示 不存在任何父節點

function 举例

1. 展開全部節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.expandAll(true);
