Function(treeNode)zTreeObj.getNodeIndex

概述[ 依赖 jquery.ztree.core 核心 js ]

獲取某節點在同級節點中的序號（從0開始）

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

需要查询順序的節點 JSON 資料對象

請務必保證此節點資料對象 是 zTree 內部的資料對象

返回值Number

返回值從 0 開始計數

如果不存在該節點資料，返回 -1

function 举例

1. 獲取當前選中的第一個節點在同級節點中的序號

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	var index = treeObj.getNodeIndex(nodes[0]);
}
