Booleansetting.edit.enable

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

設置 zTree 是否處於編輯狀態

請在初始化之前設置，初始化後需要改變編輯狀態請使用 zTreeObj.setEditable() 方法

默認值: false

Boolean 格式說明

true / false 分別表示 可以 / 不可以 編輯

編輯狀態规则說明

1、點擊節點時，不会打開 node.url 指定的 URL。
2、全面支持 編輯 與 異步加載 狀態共存。
3、可以對節點進行拖拽，且支持多棵树之間進行拖拽。
4、支持拖拽時 复制/移動 節點。（参考: setting.edit.drag.isCopy / setting.edit.drag.isMove）
5、可以通過編輯按鈕修改 name 属性。
6、可以通過删除按鈕删除節點。

請注意大小寫，不要改變

setting 举例

1. 設置 zTree 進入編輯狀態

var setting = {
	edit: {
		enable: true
	}
};
