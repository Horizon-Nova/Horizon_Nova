Function(editable)zTreeObj.setEditable

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

設置 zTree 進入 / 取消 編輯狀態。

對於編輯狀態的各種功能需要提前設置對應 setting 中的不同属性

請通過 zTree 對象執行此方法。

Function 參數說明

editableBoolean

true 表示進入 編輯狀態

false 表示取消 編輯狀態

返回值无

目前无任何返回值

function 举例

1. 設置 zTree 進入編輯狀態

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.setEditable(true);
