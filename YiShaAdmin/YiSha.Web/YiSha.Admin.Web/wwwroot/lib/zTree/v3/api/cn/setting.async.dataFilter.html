Function(treeId, parentNode, responseData)setting.async.dataFilter

概述[ 依赖 jquery.ztree.core 核心 js ]

用於對 Ajax 返回資料進行預處理的函數。[setting.async.enable = true 時生效]

默認值：null

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

parentNodeJSON

進行異步加載的父節點 JSON 資料對象

對根進行異步加載時，parentNode = null

responseDataArray(JSON) / JSON / String

異步加載獲取到的資料转換後的 Array(JSON) / JSON / String 資料對象

v3.4開始 支持 XML 資料格式的 String

返回值Array(JSON) / JSON

返回值是 zTree 支持的JSON 資料結構即可。

v3.x 支持單個 JSON 節點資料進行加載

setting & function 举例

1. 修改異步獲取到的節點name属性

function ajaxDataFilter(treeId, parentNode, responseData) {
    if (responseData) {
      for(var i =0; i < responseData.length; i++) {
        responseData[i].name += "_filter";
      }
    }
    return responseData;
};
var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		dataFilter: ajaxDataFilter
	}
};
