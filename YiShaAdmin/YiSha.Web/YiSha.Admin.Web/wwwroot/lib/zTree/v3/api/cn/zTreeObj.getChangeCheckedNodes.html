Function()zTreeObj.getChangeCheckedNodes

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

獲取輸入框勾選狀態被改變的節點集合（與原始資料 checkedOld 對比）。[setting.check.enable = true 時有效]

請通過 zTree 對象執行此方法。

Function 參數說明

返回值Array(JSON)

返回全部勾選狀態被改變的節點集合 Array

如果需要獲取每次操作後全部被改變勾選狀態的節點資料，請在每次勾選操作後，遍历所有被改變勾選狀態的節點資料，讓其 checkedOld = checked 就可以了。

function 举例

1. 獲取當前勾選狀態被改變的節點集合

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getChangeCheckedNodes();
