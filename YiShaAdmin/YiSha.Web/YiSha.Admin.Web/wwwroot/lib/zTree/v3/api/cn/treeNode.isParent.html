BooleantreeNode.isParent

概述[ 依赖 jquery.ztree.core 核心 js ]

記錄 treeNode 節點是否為父節點。

1、初始化節點資料時，根据 treeNode.children 属性判斷，有子節點则設置為 true，否则為 false

2、初始化節點資料時，如果設定 treeNode.isParent = true，即使无子節點資料，也会設置為父節點

3、為了解决部分朋友生成 json 資料出現的兼容問題, 支持 "false","true" 字符串格式的資料

Boolean 格式說明

true 表示是父節點

false 表示不是父節點

treeNode 举例

1. 查看當前被選中的節點是否是父節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isParent = sNodes[0].isParent;
}
