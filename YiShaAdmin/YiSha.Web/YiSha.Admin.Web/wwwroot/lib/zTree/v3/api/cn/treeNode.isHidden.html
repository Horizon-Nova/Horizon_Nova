BooleantreeNode.isHidden

概述[ 依赖 jquery.ztree.exhide 扩展 js ]

判斷 treeNode 節點是否被隱藏。

1、初始化 zTree 時，如果節點設置 isHidden = true，会被自動隱藏

2、請勿對已加載的節點修改此属性，隱藏 / 顯示 請使用 hideNode() / hideNodes() / showNode() / showNodes() 方法

Boolean 格式說明

true 表示被隱藏

false 表示被顯示

treeNode 举例

1. 查看第一個根節點是否被隱藏

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getNodes();
if (sNodes.length > 0) {
	var isHidden = sNodes[0].isHidden;
}
