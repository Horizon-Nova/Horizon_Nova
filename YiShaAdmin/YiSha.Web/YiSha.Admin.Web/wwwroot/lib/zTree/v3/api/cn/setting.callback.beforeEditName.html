Function(treeId, treeNode)setting.callback.beforeEditName

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用於捕獲節點編輯按鈕的 click 事件，並且根据返回值確定是否允許進入名稱編輯狀態

此事件回調函數最主要是用於捕獲編輯按鈕的點擊事件，然後触發自定義的編輯界面操作。

默認值：null

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

將要進入編輯名稱狀態的節點 JSON 資料對象

返回值Boolean

返回值是 true / false

如果返回 false，節點將无法進入 zTree 默認的編輯名稱狀態

setting & function 举例

1. 禁止修改父節點的名稱

function zTreeBeforeEditName(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeEditName: zTreeBeforeEditName
	}
};
