String / Function(treeId, treeNode)setting.edit.removeTitle

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

删除按鈕的 Title 辅助信息。[setting.edit.enable = true & setting.edit.showRemoveBtn = true 時生效]

默認值："remove"

String 格式說明

鼠標移動到 删除按鈕 上時，瀏覽器自動弹出的辅助信息內容，可根据使用者需要自行修改

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

需要設置删除按鈕 Title 信息的節點 JSON 資料對象

返回值String

返回值同 String 格式的資料

setting & function 举例

1. 設置删除按鈕的 Title 辅助信息為: "删除節點"

var setting = {
	edit: {
		enable: true,
		showRemoveBtn: true,
		removeTitle: "删除節點"
	}
};
......

2. 設置父節點删除按鈕的 Title 辅助信息為: "删除父節點"

function setRemoveTitle(treeId, treeNode) {
	return treeNode.isParent ? "删除父節點":"删除叶子節點";
}
var setting = {
	edit: {
		enable: true,
		showRemoveBtn: true,
		removeTitle: setRemoveTitle
	}
};
......
