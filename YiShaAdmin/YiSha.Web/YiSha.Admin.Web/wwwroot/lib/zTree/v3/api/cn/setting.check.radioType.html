Stringsetting.check.radioType

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

radio 的分組範圍。[setting.check.enable = true 且 setting.check.chkStyle = "radio" 時生效]

默認值："level"

String 格式說明

radioType = "level" 時，在每一級節點範圍內當做一個分組。
radioType = "all" 時，在整棵树範圍內當做一個分組。

請注意大小寫，不要改變

setting 举例

1. 設置 radio 的判別规则為整棵树內

var setting = {
	check: {
		enable: true,
		chkStyle: "radio",
		radioType: "all"
	}
};
