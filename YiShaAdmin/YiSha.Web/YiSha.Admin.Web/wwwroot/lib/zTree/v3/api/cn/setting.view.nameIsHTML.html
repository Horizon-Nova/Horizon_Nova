Booleansetting.view.nameIsHTML

概述[ 依赖 jquery.ztree.core 核心 js ]

設置 name 属性是否支持 HTML 脚本

如果允許 HTML 脚本，請根据自己的需求做校驗，避免出現 js 注入等安全問題。

默認值: false

Boolean 格式說明

true / false 分別表示 支持 / 不支持 HTML 脚本

setting 举例

1. 設置 name 属性支持 HTML 脚本

var setting = {
	view: {
		nameIsHTML: true
	}
};
var node = {"name":"<font color='red'>test</font>"};
