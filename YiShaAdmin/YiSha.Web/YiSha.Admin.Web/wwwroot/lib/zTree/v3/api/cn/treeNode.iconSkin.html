StringtreeNode.iconSkin

概述[ 依赖 jquery.ztree.core 核心 js ]

節點自定義圖標的 className

1、需要修改 css，增加相應 className 的設置

2、css 方式簡單、方便，並且同時支持父節點展開、折叠狀態切換圖片

3、css 建议采用圖片分割渲染的方式以减少反复加載圖片，並且避免圖片闪動

4、zTree v3.x 的 iconSkin 同樣支持 IE6

5、如果想直接使用 圖片的Url路徑 設置節點的個性化圖標，需要設置 treeNode.icon / treeNode.iconOpen / treeNode.iconClose 属性

默認值：无

String 格式說明

設置個性圖標的 className

css & treeNode 举例

1. 設置節點的個性化圖標

css 內容：
.ztree li span.button.diy01_ico_open, .ztree li span.button.diy01_ico_close{...}

.ztree li span.button.diy02_ico_open{...}
.ztree li span.button.diy02_ico_close{...}

.ztree li span.button.diy03_ico_docu{...}

js中節點資料：
var nodes = [
	//父節點展開 折叠時使用相同的圖標
	{ name:"父節點1", iconSkin:"diy01"},

	//父節點展開 折叠時分別使用不同的圖標
	{ name:"父節點2", iconSkin:"diy02"},

	//叶子節點個性化圖標
	{ name:"叶子節點", iconSkin:"diy03"}
]
