Stringsetting.async.dataType

概述[ 依赖 jquery.ztree.core 核心 js ]

Ajax 獲取的資料類型。[setting.async.enable = true 時生效]

默認值："text"

String 格式說明

dataType = "text" 可以滿足绝大部分請求

其余 dataType 類型請参考 jQuery ajax 中的 dataType 參數

setting 举例

1. 設置 Ajax 獲取的資料類型為 纯文本

var setting = {
	async: {
		enable: true,
		dataType: "text",
		url: "http://host/getNode.php",
		autoParam: ["id", "name"]
	}
};
