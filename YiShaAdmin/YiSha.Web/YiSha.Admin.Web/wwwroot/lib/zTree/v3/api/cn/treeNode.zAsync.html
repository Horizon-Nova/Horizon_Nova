BooleantreeNode.zAsync

概述[ 依赖 jquery.ztree.core 核心 js ]

記錄 treeNode 節點是否已经進行過異步加載，避免父節點反复異步加載資料。

初始化節點資料時，由 zTree 增加此属性，請勿提前赋值

默認值：false （无子節點的父節點）； true （有子節點的父節點 & 叶子節點）

Boolean 格式說明

true 表示父節點展開時不需要自動異步加載

false 表示父節點展開時需要自動異步加載

此參數不会對 reAsyncChildNodes 方法造成任何影响

treeNode 举例

1. 查看當前被選中的節點是否節點是否需要自動異步加載

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var zAsync = sNodes[0].zAsync;
}
