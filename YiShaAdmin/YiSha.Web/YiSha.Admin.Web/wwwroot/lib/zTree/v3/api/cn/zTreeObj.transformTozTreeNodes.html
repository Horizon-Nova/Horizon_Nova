Function(simpleNodes)zTreeObj.transformTozTreeNodes

概述[ 依赖 jquery.ztree.core 核心 js ]

將簡單 Array 格式資料转換為 zTree 使用的標准 JSON 嵌套資料格式。

使用此方法，請務必設置節點唯一標識属性名稱 setting.data.simpleData.idKey 和 父節點唯一標識属性名稱 setting.data.simpleData.pIdKey，並且讓資料滿足父子關系。

請通過 zTree 對象執行此方法。

Function 參數說明

simpleNodesArray(JSON) / JSON

需要被转換的簡單 Array 格式資料 或 某個單独的資料對象

返回值Array(JSON)

zTree 使用的標准資料，子節點都存在於父節點資料的 children 属性中

如果 simpleNodes 是一個 JSON 對象，则被簡單封装成长度為 1 的數組。

function 举例

1. 將簡單 Array 格式转換為zTree使用的標准格式

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
var simpleNodes = [
    {"id":1, "pId":0, "name":"test1"},
    {"id":11, "pId":1, "name":"test11"},
    {"id":12, "pId":1, "name":"test12"},
    {"id":111, "pId":11, "name":"test111"}
];
var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.transformTozTreeNodes(simpleNodes);
