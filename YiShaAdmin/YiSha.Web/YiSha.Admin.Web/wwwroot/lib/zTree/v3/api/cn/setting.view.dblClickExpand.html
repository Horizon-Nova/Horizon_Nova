Boolean / Function(treeId, treeNode)setting.view.dblClickExpand

概述[ 依赖 jquery.ztree.core 核心 js ]

双擊節點時，是否自動展開父節點的標識

默認值: true

Boolean 格式說明

true / false 分別表示双擊節點 切換 / 不切換 展開狀態

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

需要設置是否双擊切換展開狀態的節點 JSON 資料對象

返回值Boolean

返回值同 Boolean 格式的資料

setting 举例

1. 取消默認双擊展開父節點的功能

var setting = {
	view: {
		dblClickExpand: false
	}
};
......

2. 設置 zTree 僅僅 level=0 的父節點取消双擊展開的功能

function dblClickExpand(treeId, treeNode) {
	return treeNode.level > 0;
};
var setting = {
	view: {
		dblClickExpand: dblClickExpand
	}
};
......
