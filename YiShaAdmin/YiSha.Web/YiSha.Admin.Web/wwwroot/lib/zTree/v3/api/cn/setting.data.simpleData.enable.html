Booleansetting.data.simpleData.enable

概述[ 依赖 jquery.ztree.core 核心 js ]

確定 zTree 初始化時的節點資料、異步加載時的節點資料、或 addNodes 方法中輸入的 newNodes 資料是否采用簡單資料模式 (Array)

不需要使用者再把資料庫中取出的 List 强行转換為复杂的 JSON 嵌套格式

默認值：false

Boolean 格式說明

true / false 分別表示 使用 / 不使用 簡單資料模式

如果設置為 true，請務必設置 setting.data.simpleData 內的其他參數: idKey / pIdKey / rootPId，並且讓資料滿足父子關系。

setting 举例

1. 使用簡單 Array 格式的資料

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
var treeNodes = [

    {"id":111, "pId":11, "name":"test111"}
