Function(filter, isSingle, parentNode, invokeParam)zTreeObj.getNodesByFilter

概述[ 依赖 jquery.ztree.core 核心 js ]

根据自定義规则查詢節點資料 JSON 對象集合 或 單個節點資料

可自定義复杂的查詢规则

請通過 zTree 對象執行此方法。

Function 參數說明

filterFunction

自定義過滤器函數 function filter(node) {...}

filter 參數：node (節點資料 JSON)

filter 返回值：boolean (true 表示符合查詢條件；false 表示不符合查詢條件)

isSingleBoolean

isSingle = true 表示只查找單個節點

isSingle = false 表示查找節點集合

忽略此參數，表示查找節點集合

parentNodeJSON

可以指定在某個父節點下的子節點中查詢

忽略此參數，表示在全部節點中查詢

invokeParam任意類型

使用者自定義的資料對象，用於 filter 中進行計算

返回值Array(JSON) / JSON

isSingle = true 返回 第一個找到的節點資料 JSON，无結果時返回 null

isSingle = false 返回 節點資料集合 Array(JSON)，无結果時返回 [ ]

function 举例

1. 查找 level = 2 & name 中包含 "test" 的節點資料

function filter(node) {
    return (node.level == 2 && node.name.indexOf("test")>-1);
}
......
var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodesByFilter(filter, true); // 僅查找一個節點
var nodes = treeObj.getNodesByFilter(filter); // 查找節點集合
