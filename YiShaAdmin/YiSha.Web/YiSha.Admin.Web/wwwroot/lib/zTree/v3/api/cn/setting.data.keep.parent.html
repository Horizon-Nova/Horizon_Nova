Booleansetting.data.keep.parent

概述[ 依赖 jquery.ztree.core 核心 js ]

zTree 的節點父節點属性锁，是否始終保持 isParent = true

默認值：false

Boolean 格式說明

true / false 分別表示 锁定 / 不锁定 父節點属性

如果設置為 true，则所有 isParent = true 的節點，即使該節點的子節點被全部删除或移走，依舊保持父節點狀態。

setting 举例

1. 需要锁定父節點狀態

var setting = {
	data: {
		keep: {
			parent: true
		}
	}
};
