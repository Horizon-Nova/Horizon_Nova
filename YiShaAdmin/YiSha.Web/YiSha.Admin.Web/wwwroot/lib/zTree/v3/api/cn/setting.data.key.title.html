Stringsetting.data.key.title

概述[ 依赖 jquery.ztree.core 核心 js ]

zTree 節點資料保存節點提示信息的属性名稱。[setting.view.showTitle = true 時生效]

如果設置為 "" ，则自動與 setting.data.key.name 保持一致，避免使用者反复設置

默認值：""

setting 举例

1. 設置 zTree 顯示節點時，將 treeNode 的 fullName 属性當做節點名稱

var setting = {
	data: {
		key: {
			title: "fullName"
		}
	}
};
