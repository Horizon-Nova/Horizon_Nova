JSON / Function(treeId, treeNode)setting.view.fontCss

概述[ 依赖 jquery.ztree.core 核心 js ]

個性化文字樣式，只針對 zTree 在節點上顯示的<A>對象。

默認值：{}

JSON 格式說明

JSON 格式為 JQuery css方法中的 JSON 對象格式，例如：{color:"#ff0011", background:"blue"}

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

需要設置自定義樣式的節點 JSON 資料對象

返回值JSON

返回值同 JSON 格式的資料，例如：{color:"#ff0011", background:"blue"}

setting & function 举例

1. 不修改CSS，設置全部節點 name 顯示為红色

var setting = {
	view: {
		fontCss : {color:"red"}
	}
};

2. 設置 level=0 的節點 name 顯示為红色

function setFontCss(treeId, treeNode) {
	return treeNode.level == 0 ? {color:"red"} : {};
};
var setting = {
	view: {
		fontCss: setFontCss
	}
};
