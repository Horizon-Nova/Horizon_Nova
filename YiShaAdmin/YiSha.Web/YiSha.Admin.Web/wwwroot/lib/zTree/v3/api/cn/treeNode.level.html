NumbertreeNode.level

概述[ 依赖 jquery.ztree.core 核心 js ]

記錄節點的層級

初始化節點資料時，由 zTree 增加此属性，請勿提前赋值

Number 格式說明

根節點 level = 0，依次递增

treeNode 举例

1. 查看當前被選中的節點的級數

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var level = sNodes[0].level;
}
