Function(event, treeId, treeNodes)setting.callback.onDragMove

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用於捕獲節點被拖拽過程中移動的事件回調函數

主要用於捕獲 zTree 節點拖拽到的 DOM，從而操作對應的 DOM。

默認值：null

Function 參數說明

eventjs event 對象

標准的 js event 對象

treeIdString

被拖拽的節點 treeNodes 所在 zTree 的 treeId，便於使用者操控

treeNodesArray(JSON)

要被拖拽的節點 JSON 資料集合

setting & function 举例

1. 拖拽節點時，随時輸出 當前拖拽到的目標 DOM

function zTreeOnDragMove(event, treeId, treeNodes) {
    console.log(event.target);
};
var setting = {
	callback: {
		onDragMove: zTreeOnDragMove
	}
};
