StringtreeNode.iconOpen

概述[ 依赖 jquery.ztree.core 核心 js ]

父節點自定義展開時圖標的 URL 路徑。

1、此属性只針對父節點有效

2、此属性必須與 iconClose 同時使用

3、如果想利用 className 設置個性化圖標，需要設置 treeNode.iconSkin 属性

默認值：无

String 格式說明

圖標圖片的 url 可以是相對路徑也可以是绝對路徑

設置相對路徑請注意頁面與圖片之間的關系，確保圖片能够正常加載

treeNode 举例

1. 設置節點的個性化圖標

var nodes = [
	//父節點展開 折叠時使用相同的圖標
	{ name:"父節點1", icon:"/img/parent.gif"},

	//父節點展開 折叠時分別使用不同的圖標
	{ name:"父節點2", iconOpen:"/img/open.gif", iconClose:"/img/close.gif"}, 

	//叶子節點個性化圖標
	{ name:"叶子節點", icon:"/img/leaf.gif"}
]
