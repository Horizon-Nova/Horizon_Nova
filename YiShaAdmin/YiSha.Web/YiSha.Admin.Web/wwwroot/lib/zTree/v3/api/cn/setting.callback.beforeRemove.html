Function(treeId, treeNode)setting.callback.beforeRemove

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用於捕獲節點被删除之前的事件回調函數，並且根据返回值確定是否允許删除操作

默認值：null

Function 參數說明

treeIdString

對應 zTree 的 treeId，便於使用者操控

treeNodeJSON

將要删除的節點 JSON 資料對象

返回值Boolean

返回值是 true / false

如果返回 false，zTree 將不删除節點，也无法触發 onRemove 事件回調函數

setting & function 举例

1. 禁止全部删除操作

function zTreeBeforeRemove(treeId, treeNode) {
	return false;
}
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeRemove: zTreeBeforeRemove
	}
};
