Function(parentNode, newNodes, isSilent)zTreeObj.addNodes

概述[ 依赖 jquery.ztree.core 扩展 js ]

添加節點。

v3.x 為了避免原先反复初始化造成的資料重复問題，在 初始化 和 添加節點 時內部進行 clone 操作。如果需要獲得資料在 zTree 內的對象，請獲取此方法的返回值。

請通過 zTree 對象執行此方法。

Function 參數說明

parentNodeJSON

指定的父節點，如果增加根節點，請設置 parentNode 為 null 即可。

請務必保證此節點資料對象 是 zTree 內部的資料對象

newNodesJSON / Array(JSON)

需要增加的節點資料 JSON 對象集合，資料只需要滿足 zTree 的節點資料必需的属性即可，詳细請参考“treeNode 節點資料詳解”

1、v3.x 支持單独添加一個節點，即如果只新增一個節點，不用必須包在數組中

2、使用簡單資料模式，請参考 setting.data.simpleData 內的属性說明

isSilentBoolean

設定增加節點後是否自動展開父節點。

isSilent = true 時，不展開父節點，其他值或缺省狀態都自動展開。

返回值Array(JSON)

返回值是 zTree 最終添加的節點資料集合

如果 newNodes 是單個節點資料 JSON，返回值也是將其包在 Array 內

請務必記住：返回值中的資料對象 是 newNodes 被 clone 後的，所以绝對不相等！

function 举例

1. 對於 id = "tree" 的 zTree 增加 1 個根節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var newNode = {name:"newNode1"};
newNode = treeObj.addNodes(null, newNode);

2. 對於 id = "tree" 的 zTree 增加 3 個根節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var newNodes = [{name:"newNode1"}, {name:"newNode2"}, {name:"newNode3"}];
newNodes = treeObj.addNodes(null, newNodes);
