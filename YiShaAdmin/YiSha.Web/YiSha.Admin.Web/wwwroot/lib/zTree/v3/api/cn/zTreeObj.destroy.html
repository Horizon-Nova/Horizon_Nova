Function(treeId)zTreeObj.destroy

概述[ 依赖 jquery.ztree.core 核心 js ]

從 zTree v3.4 開始提供銷毁 zTree 的方法。

1、用此方法可以銷毁 zTreeObj 代表的 zTree。

2、銷毁當前頁面全部的 zTree，也可以使用 $.fn.zTree.destroy() 方法。

3、重新使用已经被銷毁的树，必須要使用 init 方法進行初始化。

Function Parameter Descriptions

返回值无

目前无任何返回值

function 举例

1. 銷毁 id 為 "treeDemo" 的 zTree

var zTreeObj = $.fn.zTree.getZTreeObj("treeDemo");
zTreeObj.destroy();
