Function()treeNode.getNextNode

概述[ 依赖 jquery.ztree.core 核心 js ]

獲取與 treeNode 節點相邻的後一個節點。

初始化節點資料時，由 zTree 增加此属性，請勿提前赋值

Function 參數說明

返回值JSON

與 treeNode 節點相邻的後一個節點。

如果 treeNode 是最後一個節點，返回 null 。

treeNode 举例

1. 獲取當前被選中的節點的下一個節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var node = sNodes[0].getNextNode();
}
