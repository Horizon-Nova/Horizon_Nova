BooleantreeNode.isFirstNode

概述[ 依赖 jquery.ztree.core 核心 js ]

記錄 treeNode 節點是否為同級節點中的第一個節點。

使用 exhide 扩展後，只針對顯示的節點設置此属性

初始化節點資料時，由 zTree 增加此属性，請勿提前赋值

Boolean 格式說明

true 表示是同級節點中的第一個節點

false 表示不是同級節點中的第一個節點

節點被隱藏後，isFirstNode = false

treeNode 举例

1. 查看當前被選中的節點是否是同級節點中的第一個節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isFirstNode = sNodes[0].isFirstNode;
}
