StringtreeNode.parentTId

概述[ 依赖 jquery.ztree.core 核心 js ]

treeNode 節點的父節點唯一標識 tId。

1、v3.x 用 parentTId 替換了原先的 parentNode 属性，同時增加了 getParentNode 方法，以避免原先 parentNode 造成的 clone 死循环

2、初始化節點資料時，由 zTree 增加此属性，請勿提前赋值

String 格式說明

zTree 內部生成的節點唯一標識，請参考 treeNode.tId 的說明

如果 treeNode 是根節點，则 parentTId = null

treeNode 举例

1. 查看當前被選中的節點的父節點 tId

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var parentTId = sNodes[0].parentTId;
}
