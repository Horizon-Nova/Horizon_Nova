Function(treeId, treeNodes, targetNode, moveType, isCopy)setting.callback.beforeDrop

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用於捕獲節點拖拽操作結束之前的事件回調函數，並且根据返回值確定是否允許此拖拽操作

默認值：null

如未拖拽到有效位置，则不触發此回調函數，直接將節點恢复原位置

Function 參數說明

treeIdString

目標節點 targetNode 所在 zTree 的 treeId，便於使用者操控

treeNodesArray(JSON)

被拖拽的節點 JSON 資料集合

无论拖拽操作為 复制 還是 移動，treeNodes 都是當前被拖拽節點的資料集合。

targetNodeJSON

treeNodes 被拖拽放開的目標節點 JSON 資料對象。

如果拖拽成為根節點，则 targetNode = null

moveTypeString

指定移動到目標節點的相對位置

"inner"：成為子節點，"prev"：成為同級前一個節點，"next"：成為同級後一個節點

isCopyBoolean

拖拽節點操作是 复制 或 移動

true：复制；false：移動

返回值Boolean

返回值是 true / false

如果返回 false，zTree 將恢复被拖拽的節點，也无法触發 onDrop 事件回調函數

setting & function 举例

1. 禁止將節點拖拽成為根節點

function zTreeBeforeDrop(treeId, treeNodes, targetNode, moveType) {
    return !(targetNode == null || (moveType != "inner" && !targetNode.parentTId));
};
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeDrop: zTreeBeforeDrop
	}
};
