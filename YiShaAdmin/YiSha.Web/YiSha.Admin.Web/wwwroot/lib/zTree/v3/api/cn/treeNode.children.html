Array(JSON)treeNode.children

概述[ 依赖 jquery.ztree.core 核心 js ]

節點的子節點資料集合。

1、如果不使用 children 属性保存子節點資料，請修改 setting.data.key.children

2、異步加載時，對於設置了 isParent = true 的節點，在展開時將進行異步加載

默認值：无

Array(JSON) 格式說明

標准的 JSON 資料對象

treeNode 举例

1. 初始化的標准嵌套格式的 JSON 資料對象

var nodes = [
{ "id":1, "name":"test1",
	children: [
	{ "id":3, "name":"test3"},
	{ "id":4, "name":"test4"},
	{ "id":5, "name":"test5"}
	]
},
{ "id":2, "name":"test2"  }
]

2. 獲取第一個根節點的子節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes()[0].children;
