Function(treeNode)zTreeObj.hideNode

概述[ 依赖 jquery.ztree.exhide 扩展 js ]

隱藏某個節點。

1、此功能不支持 exedit 扩展，因此不要在編輯狀態時使用隱藏節點的方法。

2、隱藏/顯示節點，会影响節點的 isFirstNode 和 isLastNode 属性。

3、請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

指定被隱藏的節點 JSON 資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

返回值无

目前无任何返回值

function 举例

1. 隱藏根節點第一個節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.hideNode(nodes[0]);
