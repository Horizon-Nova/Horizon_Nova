StringtreeNode.name

概述[ 依赖 jquery.ztree.core 核心 js ]

節點名稱。

1、如果不使用 name 属性保存節點名稱，請修改 setting.data.key.name

默認值：无

String 格式說明

節點顯示的名稱字符串，標准 String 即可，所有特殊字符都会被自動转義

treeNode 举例

1. 設置節點的名稱為 test1、test2、test3

var nodes = [
	{ "id":1, "name":"test1"},
	{ "id":2, "name":"test2"},
	{ "id":3, "name":"test3"}
]
