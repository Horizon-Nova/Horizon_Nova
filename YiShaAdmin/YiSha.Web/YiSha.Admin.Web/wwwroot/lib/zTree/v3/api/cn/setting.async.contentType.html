Stringsetting.async.contentType

概述[ 依赖 jquery.ztree.core 核心 js ]

Ajax 提交參數的資料類型。[setting.async.enable = true 時生效]

默認值："application/x-www-form-urlencoded"

String 格式說明

contentType = "application/x-www-form-urlencoded" 可以滿足绝大部分請求，按照標准的 Form 格式提交參數

contentType = "application/json" 可以滿足 .Net 的編程需要，按照 JSON 格式提交參數

setting 举例

1. 設置 Ajax 提交參數的資料類型為 JSON 格式

var setting = {
	async: {
		enable: true,
		contentType: "application/json",
		url: "http://host/getNode.php",
		autoParam: ["id", "name"]
	}
};
