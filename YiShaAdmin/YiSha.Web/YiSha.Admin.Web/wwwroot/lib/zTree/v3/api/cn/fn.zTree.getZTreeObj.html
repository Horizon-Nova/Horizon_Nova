Function(treeId)$.fn.zTree.getZTreeObj

概述[ 依赖 jquery.ztree.core 核心 js ]

zTree v3.x 專門提供的根据 treeId 獲取 zTree 對象的方法。

必須在初始化 zTree 以後才可以使用此方法。

有了這個方法，使用者不再需要自己設定全局變量來保存 zTree 初始化後得到的對象了，而且在所有回調函數中全都会返回 treeId 属性，使用者可以随時使用此方法獲取需要進行操作的 zTree 對象

Function 參數說明

treeIdString

zTree 的 DOM 容器的 id

返回值JSON

zTree 對象，提供操作 zTree 的各種方法，對於通過 js 操作 zTree 來說必須通過此對象

function 举例

1. 獲取 id 為 tree 的 zTree 對象

var treeObj = $.fn.zTree.getZTreeObj("tree");
