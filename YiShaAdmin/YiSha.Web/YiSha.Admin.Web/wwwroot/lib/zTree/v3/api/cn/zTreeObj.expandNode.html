Function(treeNode, expandFlag, sonSign, focus, callbackFlag)zTreeObj.expandNode

概述[ 依赖 jquery.ztree.core 核心 js ]

展開 / 折叠 指定的節點

v3.x 中執行此方法可以触發 beforeExpand / onExpand 或 beforeCollapse / onCollapse 事件回調函數。便於减少冗余代碼

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

需要 展開 / 折叠 的節點資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

expandFlagBoolean

expandFlag = true 表示 展開 節點

expandFlag = false 表示 折叠 節點

省略此參數，则根据對此節點的展開狀態進行 toggle 切換

sonSignBoolean

sonSign = true 表示 全部子孙節點 進行與 expandFlag 相同的操作

sonSign = false 表示 只影响此節點，對於其 子孙節點无任何影响

sonSign = false 且 treeNode.open = expandFlag 時，不会触發回調函數，直接返回

省略此參數，等同於 false

focusBoolean

focus = true 表示 展開 / 折叠 操作後，通過設置焦點保證此焦點進入可視區域內

focus = false 表示 展開 / 折叠 操作後，不設置任何焦點

省略此參數，等同於 true

callbackFlagBoolean

callbackFlag = true 表示執行此方法時触發 beforeExpand / onExpand 或 beforeCollapse / onCollapse 事件回調函數

callbackFlag = false 表示執行此方法時不触發事件回調函數

省略此參數，等同於 false

返回值Boolean

返回值表示最終實際操作情况

true 表示 展開 節點

false 表示 折叠 節點

null 表示 不是父節點

function 举例

1. 展開當前選擇的第一個節點（包括其全部子節點）

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	treeObj.expandNode(nodes[0], true, true, true);
}
