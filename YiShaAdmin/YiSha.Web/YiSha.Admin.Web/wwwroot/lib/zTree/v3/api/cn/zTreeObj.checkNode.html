Function(treeNode, checked, checkTypeFlag, callbackFlag)zTreeObj.checkNode

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

勾選 或 取消勾選 單個節點。[setting.check.enable = true 時有效]

v3.x 中 checkNode() 方法可以触發 beforeCheck / onCheck 事件回調函數。便於减少冗余代碼

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

需要勾選 或 取消勾選 的節點資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

checkedBoolean

checked = true 表示勾選節點

checked = false 表示節點取消勾選

省略此參數，则根据對此節點的勾選狀態進行 toggle 切換

不影响 treeNode.nochecked = true 的節點。

checkTypeFlagBoolean

checkTypeFlag = true 表示按照 setting.check.chkboxType 属性進行父子節點的勾選联動操作

checkTypeFlag = false 表示只修改此節點勾選狀態，无任何勾選联動操作

checkTypeFlag = false 且 treeNode.checked = checked 時，不会触發回調函數，直接返回

不影响父子節點中 treeNode.nochecked = true 的節點。

callbackFlagBoolean

callbackFlag = true 表示執行此方法時触發 beforeCheck & onCheck 事件回調函數

callbackFlag = false 表示執行此方法時不触發事件回調函數

省略此參數，等同於 false

返回值无

目前无任何返回值

function 举例

1. 勾選當前選中的節點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
for (var i=0, l=nodes.length; i < l; i++) {
	treeObj.checkNode(nodes[i], true, true);
}
