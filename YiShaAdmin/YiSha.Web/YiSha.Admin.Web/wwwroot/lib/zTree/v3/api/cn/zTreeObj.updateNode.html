Function(treeNode, checkTypeFlag)zTreeObj.updateNode

概述[ 依赖 jquery.ztree.core 核心 js ]

更新某節點資料，主要用於該節點顯示属性的更新。

1、可針對 name、target、 url、icon、 iconSkin、checked、nocheck 等這几個用於顯示效果的參數進行更新，其他用於 zTreeNodes 的參數請不要随意更新，對於展開節點，還請調用 expandNode方法，因此請勿随意修改 open 属性。

2、用此方法修改 checked 勾選狀態不会触發 beforeCheck / onCheck 事件回調函數。

請通過 zTree 對象執行此方法。

Function 參數說明

treeNodeJSON

指定需要更新的節點 JSON 資料

請務必保證此節點資料對象 是 zTree 內部的資料對象

checkTypeFlagBoolean

checkTypeFlag = true 表示按照 setting.check.chkboxType 属性進行父子節點的勾選联動操作

checkTypeFlag = false 表示只修改此節點勾選狀態，无任何勾選联動操作

當 setting.check.enable = true 且 setting.check.chkStyle = "checkbox" 時才有效

不影响父子節點中 treeNode.nochecked = true 的節點。

返回值无

目前无任何返回值

function 举例

1. 更新根節點中第一個節點的名稱

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
if (nodes.length>0) {
	nodes[0].name = "test";
	treeObj.updateNode(nodes[0]);
}
