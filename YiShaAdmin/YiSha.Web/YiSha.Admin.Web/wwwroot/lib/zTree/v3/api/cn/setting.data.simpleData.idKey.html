Stringsetting.data.simpleData.idKey

概述[ 依赖 jquery.ztree.core 核心 js ]

節點資料中保存唯一標識的属性名稱。[setting.data.simpleData.enable = true 時生效]

默認值："id"

setting 举例

1. 使用簡單 Array 格式的資料

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
var treeNodes = [
    {"id":1, "pId":0, "name":"test1"},
    {"id":11, "pId":1, "name":"test11"},
    {"id":12, "pId":1, "name":"test12"},
    {"id":111, "pId":11, "name":"test111"}
];
