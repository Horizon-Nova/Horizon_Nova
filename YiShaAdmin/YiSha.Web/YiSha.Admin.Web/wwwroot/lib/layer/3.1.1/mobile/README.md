
## layer mobile
layer mobile是為移動設備（手機、平板等webkit內核瀏覽器/webview）量身定做的弹層支撑，采用Native JavaScript編寫，完全独立於PC版的layer，您需要按照场景選擇使用。

[檔案與演示](http://sentsin.com/layui/layer/)   

1. 无需依赖任何庫，只加載layer.m.js即可
2. 小巧玲珑，性能卓越、柔情似水…
3. 具備无以伦比的自适應功能
4. 灵活的皮肤自定義支撑，充分確保弹層风格多樣化
5. 丰富、科学的接口，讓弹弹弹層无所不能
