## YiShaAdmin
 YiShaAdmin 基於.NET Core Web開發，借鉴了很多開源項目的優點，讓你開發Web管理系统和移動端Api更簡單，所以我也把她開源了。
 她可以用於所有的Web應用程序，例如网站管理後台、CMS、CRM、ERP、OA這類的系统和移動端Api。<br />

## 內置功能

- 員工管理：員工是系统操作者，該功能主要完成系统使用者配置
- 部門管理：配置系统組织機構（公司、部門、小組）
- 職位管理：配置系统使用者所担任職務
- 文章中心：管理新闻及公司案例
- 角色管理：角色選單權限分配，角色對應權限，員工属於某個角色
- 選單管理：配置系统選單，操作權限，按鈕權限標識
- 通用字典：系统里動態改變的資料，像文章類型，用字典進行維護，不變的資料，可以用枚举
- 行政區划：全国的省市县三級資料，展示树形資料的使用
- 系统日誌（登錄日誌、操作日誌、Api日誌）：查看系统的登錄、操作、Api調用日誌
- 定時任務：在線（添加、修改、删除）任務調度，包含執行結果日誌
- 系统Api：Api檔案與测试工具
- 資料表管理：查看當前資料庫所有表、字段及記錄總數
- 代碼生成：快速生成對單表的增删改查
- 服務器信息：查看服務器的基本信息
