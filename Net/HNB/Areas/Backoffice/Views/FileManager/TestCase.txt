# FileManager 功能測試案例

## 說明
此檔案用於記錄 FileManager 的所有功能測試案例，每次開發/修正功能時需核對，確保沒有破壞現有功能。

---

| 功能 | 操作 | 期望結果 |
|------|------|----------|
| 資料夾區 | 打開畫面 | 1. 進入目錄根 2. 看到側邊欄顯示樹狀結構的資料夾列表 3. 巢狀資料夾正確顯示（如 123/1234/12345），不會平鋪成列表 4. 資料夾預設為收合狀態（顯示向右箭頭）5. 點擊箭頭可以展開/收合資料夾 |
| 儲存空間 | 打開畫面 | 1. 進入目錄根 2. 在側邊欄底部看到儲存空間資訊 3. 顯示格式為「已使用空間 / 總空間限制」（如 673.93 MB / 4.88 GB）4. 顯示進度條，正確顯示使用百分比 |
| 我的檔案-內容 | 打開畫面 | 1. 進入目錄根 2. 看到DataTable顯示檔案和資料夾列表 3. 列表包含名稱、共用、擁有者、修改日期、大小等欄位 4. 可以排序和搜尋 |
| 我的檔案-內容 | 點擊列表中的項目 | 1. 該項目行高亮顯示 2. 右側詳細資訊面板顯示該項目的詳細資訊 3. 詳細資訊包含位置、擁有者、共用、大小、建立時間、修改時間等 |
| 工具列-共享功能 | 選擇項目後點擊共用按鈕 | 1. 開啟共享Modal 2. Modal顯示項目名稱和類型 3. 顯示所有可用使用者列表 4. 已共用的使用者checkbox為勾選狀態 |
| 工具列-共享功能 | 在共享Modal中選擇使用者並提交 | 1. 顯示成功訊息 2. Modal關閉 3. 項目詳細資訊面板更新，顯示新的共用資訊 4. 檔案列表中的共用欄位更新 |
| 工具列-麵包屑 | 打開畫面 | 1. 進入目錄根 2. 看到麵包屑顯示「我的檔案」3. 統計資訊顯示「X 個資料夾 · Y 個檔案 · Z B」 |
| 工具列-麵包屑 | 點擊資料夾進入子目錄 | 1. 麵包屑更新為「我的檔案 > 資料夾名稱」2. 統計資訊更新為該資料夾的統計 3. URL更新但頁面不重載 |
| 工具列-麵包屑 | 點擊麵包屑中的「我的檔案」 | 1. 切換回根目錄 2. 麵包屑更新為「我的檔案」3. URL更新但頁面不重載 4. 顯示根目錄的檔案列表 |
| 工具列-統計資訊 | 切換不同資料夾 | 1. 統計資訊即時更新 2. 顯示該資料夾內的資料夾數量、檔案數量和總大小 3. 格式為「X 個資料夾 · Y 個檔案 · Z B」 |
| 共享Modal-搜尋下拉選單 | 在搜尋框輸入關鍵字 | 1. 輸入時即時顯示符合的使用者下拉選單 2. 下拉選單顯示使用者頭像、姓名、電子郵件 3. 已選擇的使用者顯示為disabled並標示✓ 4. 找不到符合的使用者時顯示「找不到符合的使用者」 |
| 共享Modal-搜尋下拉選單 | 點擊下拉選單中的使用者 | 1. 該使用者的checkbox自動勾選 2. 搜尋框清空 3. 下拉選單關閉 4. 已選擇的使用者顯示在上方徽章區域 |
| 共享Modal-搜尋下拉選單 | 點擊已選擇使用者徽章的刪除按鈕 | 1. 該使用者從徽章區域移除 2. 對應的checkbox取消勾選 3. 下拉選單中該使用者恢復為可選擇狀態 |
| 上傳檔案/資料夾-背景上傳 | 上傳過程中關閉Modal | 1. 上傳繼續進行不中斷 2. 右下角顯示背景進度條 3. 進度條顯示百分比和已上傳/總大小 4. 上傳完成後自動隱藏進度條並顯示成功提示 |
| 上傳檔案/資料夾-取消上傳 | 上傳過程中點擊Modal內的取消按鈕 | 1. 顯示確認對話框「上傳正在進行中，確定要取消上傳嗎？」2. 確認後中斷上傳 3. 顯示「上傳已取消」提示 4. Modal關閉 |
| 上傳檔案/資料夾-取消上傳 | 上傳過程中關閉Modal後點擊背景進度條的關閉按鈕 | 1. 顯示確認對話框「上傳正在進行中，確定要取消上傳嗎？」2. 確認後中斷上傳 3. 顯示「上傳已取消」提示 4. 背景進度條消失 |
| 上傳檔案/資料夾-檔案大小限制 | 選擇總大小超過5GB的檔案或資料夾 | 1. 選擇檔案時立即顯示錯誤提示「檔案大小超過 5GB（目前：X.XX GB），無法上傳」2. 檔案選擇被清空 3. 無法提交上傳 |

---

## 測試注意事項

1. 每次修改功能後，需核對所有相關測試案例
2. 新增功能時，需在此檔案中新增對應的測試案例
3. 修正 Bug 時，需確認相關測試案例是否通過
4. 重構程式碼時，需確保所有測試案例仍能通過
