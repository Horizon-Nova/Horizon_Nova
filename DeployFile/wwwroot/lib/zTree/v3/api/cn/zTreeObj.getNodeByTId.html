Function(tId)zTreeObj.getNodeByTId

概述[ 依赖 jquery.ztree.core 核心 js ]

根据 zTree 的唯一識別 tId 快速獲取結點 JSON 資料对象

通过内部的 cache 獲取，不需要遍历結點。

請通过 zTree 对象执行此方法。

Function 参數说明

tIdString

結點在 zTree 内的唯一識別 tId

返回值JSON

tId 对应的結點 JSON 資料对象

如无结果，返回 null

function 举例

1. 獲取 tId = "tree_10" 的結點資料

var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodeByTId("tree_10");
