Function(targetNode, treeNode, moveType, isSilent)zTreeObj.moveNode

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

移动結點。

請通过 zTree 对象执行此方法。

Function 参數说明

targetNodeJSON

要移动到的目标結點 JSON 資料

如果移动成為根結點，請设置 targetNode 為 null 即可

請务必保证此結點資料对象 是 zTree 内部的資料对象

treeNodeJSON

需要被移动的結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

moveTypeString

指定移动到目标結點的相对位置

"inner"：成為子結點，"prev"：成為同级前一个結點，"next"：成為同级後一个結點

isSilentBoolean

设定移动結點後是否自动展开父結點。

isSilent = true 时，不展开父結點，其他值或缺省状态都自动展开。

返回值JSON

返回值是最终被移动的結點資料，正常情况下与 treeNode 参數完全相同

如果 返回值 為 null，说明 移动失败，主要原因有：
 1、targetNode 是 treeNode 父結點，且 moveType = "inner"
 2、targetNode 是 treeNode 子孙結點

function 举例

1. 将根結點中第二个結點 移动成為 第一个結點的子結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.moveNode(nodes[0], nodes[1], "inner");

2. 将根結點中第二个結點 移动成為 第一个結點的前一个結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.moveNode(nodes[0], nodes[1], "prev");
