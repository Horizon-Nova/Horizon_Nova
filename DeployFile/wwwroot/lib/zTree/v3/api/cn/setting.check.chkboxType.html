JSONsetting.check.chkboxType

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

勾選 checkbox 对于父子結點的关联关系。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" 时生效]

默认值：{ "Y": "ps", "N": "ps" }

JSON 格式说明

Y 属性定义 checkbox 被勾選後的情况；
N 属性定义 checkbox 取消勾選後的情况；
"p" 表示操作會影响父級結點；
"s" 表示操作會影响子級結點。

請注意大小写，不要改变

setting 举例

1. checkbox 勾選操作，只影响父級結點；取消勾選操作，只影响子級結點

var setting = {
	check: {
		enable: true,
		chkStyle: "checkbox",
		chkboxType: { "Y": "p", "N": "s" }
	}
};
