BooleantreeNode.open

概述[ 依赖 jquery.ztree.core 核心 js ]

记录 treeNode 結點的 展开 / 折叠 状态。

1、初始化結點資料时，如果设定 treeNode.open = true，则會直接展开此結點

2、叶子結點 treeNode.open = false

3、為了解决部分朋友生成 json 資料出现的兼容问题, 支持 "false","true" 字符串格式的資料

4、非异步加載模式下，无子結點的父結點设置 open=true 後，可顯示為展开状态，但异步加載模式下不會生效。（v3.5.15+）

默认值：false

Boolean 格式说明

true 表示結點為 展开 状态

false 表示結點為 折叠 状态

treeNode 举例

1. 查看當前被選中的結點的 展开 / 折叠 状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isOpen = sNodes[0].open;
}
