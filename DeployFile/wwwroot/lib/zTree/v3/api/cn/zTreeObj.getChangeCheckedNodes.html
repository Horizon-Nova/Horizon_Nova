Function()zTreeObj.getChangeCheckedNodes

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

獲取输入框勾選状态被改变的結點集合（与原始資料 checkedOld 对比）。[setting.check.enable = true 时有效]

請通过 zTree 对象执行此方法。

Function 参數说明

返回值Array(JSON)

返回全部勾選状态被改变的結點集合 Array

如果需要獲取每次操作後全部被改变勾選状态的結點資料，請在每次勾選操作後，遍历所有被改变勾選状态的結點資料，让其 checkedOld = checked 就可以了。

function 举例

1. 獲取當前勾選状态被改变的結點集合

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getChangeCheckedNodes();
