Function(checked)zTreeObj.checkAllNodes

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

勾選 或 取消勾選 全部結點。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" 时有效]

此方法不會触发 beforeCheck / onCheck 事件回调函數。

請通过 zTree 对象执行此方法。

Function 参數说明

checkedBoolean

checked = true 表示勾選全部結點

checked = false 表示全部結點取消勾選

不會影响 treeNode.nochecked = true 的結點。

不會影响未加載的結點。

返回值无

目前无任何返回值

function 举例

1. 勾選全部結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.checkAllNodes(true);
