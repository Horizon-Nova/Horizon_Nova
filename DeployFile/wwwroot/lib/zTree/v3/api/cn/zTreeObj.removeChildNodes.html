Function(parentNode)zTreeObj.removeChildNodes

概述[ 依赖 jquery.ztree.core 扩展 js ]

清空某父結點的子結點。

1、清空子結點後，父結點會自动变為叶子結點，如需要父結點保持父結點状态，請设置 setting.data.keep.parent 属性。

2、請勿用此方法清空根結點，如果需要清空根結點，直接初始化 zTree，并且设置初始結點為 null 即可。

3、此方法不會触发任何事件回调函數。

請通过 zTree 对象执行此方法。

Function 参數说明

parentNodeJSON

需要清空子結點的父結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

返回值Array(JSON)

将该父結點的子結點資料返回，如果不存在则返回 null

function 举例

1. 清空選中的第一个結點的子結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes && nodes.length>0) {
	treeObj.removeChildNodes(nodes[0]);
}
