Function(treeNode, expandFlag, sonSign, focus, callbackFlag)zTreeObj.expandNode

概述[ 依赖 jquery.ztree.core 核心 js ]

展开 / 折叠 指定的結點

v3.x 中执行此方法可以触发 beforeExpand / onExpand 或 beforeCollapse / onCollapse 事件回调函數。便于减少冗余代码

請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

需要 展开 / 折叠 的結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

expandFlagBoolean

expandFlag = true 表示 展开 結點

expandFlag = false 表示 折叠 結點

省略此参數，则根据对此結點的展开状态进行 toggle 切换

sonSignBoolean

sonSign = true 表示 全部子孙結點 进行与 expandFlag 相同的操作

sonSign = false 表示 只影响此結點，对于其 子孙結點无任何影响

sonSign = false 且 treeNode.open = expandFlag 时，不會触发回调函數，直接返回

省略此参數，等同于 false

focusBoolean

focus = true 表示 展开 / 折叠 操作後，通过设置焦點保证此焦點进入可視區域内

focus = false 表示 展开 / 折叠 操作後，不设置任何焦點

省略此参數，等同于 true

callbackFlagBoolean

callbackFlag = true 表示执行此方法时触发 beforeExpand / onExpand 或 beforeCollapse / onCollapse 事件回调函數

callbackFlag = false 表示执行此方法时不触发事件回调函數

省略此参數，等同于 false

返回值Boolean

返回值表示最终实际操作情况

true 表示 展开 結點

false 表示 折叠 結點

null 表示 不是父結點

function 举例

1. 展开當前選擇的第一个結點（包括其全部子結點）

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	treeObj.expandNode(nodes[0], true, true, true);
}
