Array(String)setting.async.autoParam

概述[ 依赖 jquery.ztree.core 核心 js ]

异步加載时需要自动提交父結點属性的参數。[setting.async.enable = true 时生效]

默认值：[ ]

Array(String) 格式说明

1、将需要作為参數提交的属性名称，制作成 Array 即可，例如：["id", "name"]

2、可以设置提交时的参數名称，例如 server 只接受 zId : ["id=zId"]

setting 举例

1. 设置 id 属性為自动提交的参數

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id"]
	}
};
假设 异步加載 父結點(node = {id:1, name:"test"}) 的子結點时，将提交参數 id=1
......

2. 设置 id 属性作為 zId 成為自动提交的参數

var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		autoParam: ["id=zId"]
	}
};
假设 对父結點 node = {id:1, name:"test"}，进行异步加載时，将提交参數 zId=1
......
