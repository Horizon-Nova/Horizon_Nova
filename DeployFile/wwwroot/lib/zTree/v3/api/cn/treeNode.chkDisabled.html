BooleantreeNode.chkDisabled

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

1、设置結點的 checkbox / radio 是否禁用 [setting.check.enable = true 时有效]

2、為了解决部分朋友生成 json 資料出现的兼容问题, 支持 "false","true" 字符串格式的資料

3、請勿对已加載的結點修改此属性，禁止 或 取消禁止 請使用 setChkDisabled() 方法

4、初始化时，如果需要子孙結點继承父結點的 chkDisabled 属性，請设置 setting.check.chkDisabledInherit 属性

默认值：false

Boolean 格式说明

true 表示此結點的 checkbox / radio 被禁用。

false 表示此結點的 checkbox / radio 可以使用。

treeNode 举例

1. 禁用結點 checkbox / radio 

var nodes = [
	{ "id":1, "name":"test1", "checked":true, "chkDisabled":true},
	{ "id":2, "name":"test2", "chkDisabled":true},
	{ "id":3, "name":"test3"}
]
