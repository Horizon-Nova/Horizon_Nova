Function(treeId, treeNodes)setting.callback.beforeDrag

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用于捕獲結點被拖拽之前的事件回调函數，并且根据返回值确定是否允許开启拖拽操作

默认值：null

Function 参數说明

treeIdString

被拖拽的結點 treeNodes 所在 zTree 的 treeId，便于用户操控

treeNodesArray(JSON)

要被拖拽的結點 JSON 資料集合

v3.x 允許多个同级結點同时被拖拽，因此将此参數修改為 Array(JSON)

如果拖拽时多个被選擇的結點不是同级关系，则只能拖拽鼠标當前所在位置的結點

返回值Boolean

返回值是 true / false

如果返回 false，zTree 将中止拖拽，也无法触发 onDrag / beforeDrop / onDrop 事件回调函數

setting & function 举例

1. 禁止全部拖拽操作

function zTreeBeforeDrag(treeId, treeNodes) {
    return false;
};
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeDrag: zTreeBeforeDrag
	}
};
