BooleantreeNode.isLastNode

概述[ 依赖 jquery.ztree.core 核心 js ]

记录 treeNode 結點是否為同级結點中的最後一个結點。

使用 exhide 扩展後，只针对顯示的結點设置此属性

初始化結點資料时，由 zTree 增加此属性，請勿提前赋值

Boolean 格式说明

true 表示是同级結點中的最後一个結點

false 表示不是同级結點中的最後一个結點

結點被隱藏後，isLastNode = false

treeNode 举例

1. 查看當前被選中的結點是否是同级結點中的最後一个結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isLastNode = sNodes[0].isLastNode;
}
