Function(treeNode)zTreeObj.editName

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

设置某結點进入编辑名称状态。

1、如果需要用 js 取消编辑名称状态，請使用 cancelEditName(newName) 方法。

2、可利用此方法让當前正编辑的結點 input 输入框獲取焦點。

3、請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

指定进入编辑名称状态的結點 JSON 資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

返回值无

目前无任何返回值

function 举例

1. 设置根結點第一个結點进入编辑名称状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.editName(nodes[0]);
