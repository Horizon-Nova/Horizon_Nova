Function(treeNode)zTreeObj.cancelSelectedNode

概述[ 依赖 jquery.ztree.core 核心 js ]

取消結點的選中状态。

v3.x 支持多點同时選中，因此取消選中状态可以全部取消，也可以單独取消某个結點的選中状态。

請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

需要取消選中状态的結點。

請务必保证此結點資料对象 是 zTree 内部的資料对象

如果省略此参數，则将取消全部被選中結點的選中状态。

返回值无

目前无任何返回值

function 举例

1. 取消當前所有被選中結點的選中状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
treeObj.cancelSelectedNode();

2. 取消當前第一个被選中結點的選中状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) { 
	treeObj.cancelSelectedNode(nodes[0]);
}
