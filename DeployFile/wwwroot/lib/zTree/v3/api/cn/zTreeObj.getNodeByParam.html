Function(key, value, parentNode)zTreeObj.getNodeByParam

概述[ 依赖 jquery.ztree.core 核心 js ]

根据結點資料的属性搜索，獲取条件完全匹配的結點資料 JSON 对象

請通过 zTree 对象执行此方法。

Function 参數说明

keyString

需要精确匹配的属性名称

value?

需要精确匹配的属性值，可以是任何类型，只要保证与 key 指定的属性值保持一致即可

parentNodeJSON

搜索范围，指定在某个父結點下的子結點中进行搜索

忽略此参數，表示在全部結點中搜索

返回值JSON

匹配精确搜索的結點資料

1、如无结果，返回 null

2、如有多个結點满足查询条件，只返回第一个匹配到的結點

function 举例

1. 查找 id = 1 的結點資料

var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodeByParam("id", 1, null);
