BooleantreeNode.nocheck

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

1、设置結點是否隱藏 checkbox / radio [setting.check.enable = true 时有效]

2、為了解决部分朋友生成 json 資料出现的兼容问题, 支持 "false","true" 字符串格式的資料

默认值：false

Boolean 格式说明

true 表示此結點不顯示 checkbox / radio，不影响勾選的关联关系，不影响父結點的半選状态。

false 表示結點具有正常的勾選功能

treeNode 举例

1. 不顯示某个結點的 checkbox / radio 

var nodes = [
	{ "id":1, "name":"test1", "nocheck":true},
	{ "id":2, "name":"test2"},
	{ "id":3, "name":"test3"}
]
