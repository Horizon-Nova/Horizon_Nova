Function(treeId, treeNode)setting.callback.beforeEditName

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用于捕獲結點编辑按鈕的 click 事件，并且根据返回值确定是否允許进入名称编辑状态

此事件回调函數最主要是用于捕獲编辑按鈕的點擊事件，然後触发自定义的编辑界面操作。

默认值：null

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

将要进入编辑名称状态的結點 JSON 資料对象

返回值Boolean

返回值是 true / false

如果返回 false，結點将无法进入 zTree 默认的编辑名称状态

setting & function 举例

1. 禁止修改父結點的名称

function zTreeBeforeEditName(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true
	},
	callback: {
		beforeEditName: zTreeBeforeEditName
	}
};
