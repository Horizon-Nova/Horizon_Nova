Booleansetting.check.chkDisabledInherit

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

当父結點设置 chkDisabled = true 时，设置子結點是否自动继承 chkDisabled = true 。[setting.check.enable = true 时生效]

1、只使用于初始化結點时，便于批量操作。 对于已存在的結點請利用 setChkDisabled 方法單个結點设置。

默认值: false

Boolean 格式说明

true 表示 新加入子結點时，自动继承父結點 chkDisabled = true 的属性。

false 表示 新加入子結點时，不继承父結點 chkDisabled 的属性。

setting 举例

1. 需要子結點自动继承 chkDisabled = true

var setting = {
	check: {
		enable: true,
		chkDisabledInherit: true
	}
};
