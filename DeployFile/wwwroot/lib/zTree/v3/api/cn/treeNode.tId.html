StringtreeNode.tId

概述[ 依赖 jquery.ztree.core 核心 js ]

treeNode 結點的唯一識別 tId。

初始化結點資料时，由 zTree 增加此属性，請勿提前赋值

String 格式说明

生成规则：setting.treeId + "_" + 内部计數

treeNode 举例

1. 查看當前被選中的結點的 tId

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var tId = sNodes[0].tId;
}
