Booleansetting.check.autoCheckTrigger

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

设置自动关联勾選时是否触发 beforeCheck / onCheck 事件回调函數。[setting.check.enable = true 且 setting.check.chkStyle = "checkbox" 时生效]

1、如果设置 setting.check.chkboxType = { "Y": "", "N": "" }，将不會有任何自动关联勾選的操作。

2、如果开启触发，对于結點较多的树将會影响性能，因為所有被联动勾選的操作都會触发事件回调函數，請根据需要决定是否使用此功能。

默认值: false

Boolean 格式说明

true / false 分别表示 触发 / 不触发 事件回调函數

setting 举例

1. 需要触发自动关联勾選操作

var setting = {
	check: {
		enable: true,
		autoCheckTrigger: true
	}
};
