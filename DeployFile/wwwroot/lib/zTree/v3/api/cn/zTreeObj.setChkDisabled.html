Function(node, disabled, inheritParent, inheritChildren)zTreeObj.setChkDisabled

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

禁用 或 解禁 某个結點的 checkbox / radio [setting.check.enable = true 时有效]

1、結點的 checkbox / radio 被禁用後，无法勾選或取消勾選，但能够影响父結點的半選状态

2、請不要直接修改已加載結點的 treeNode.chkDisabled 属性。

3、請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

需要禁用 或 解禁 checkbox / radio 的結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

disabledBoolean

disabled = true 表示禁用 checkbox / radio

disabled = false 表示解禁 checkbox / radio

省略此参數，等同于 disabled = false

不影响 treeNode.nochecked = true 的結點。

inheritParentBoolean

inheritParent = true 表示全部父結點进行同样的操作

inheritParent = false 表示不影响父結點

省略此参數，等同于 inheritParent = false

inheritChildrenBoolean

inheritChildren = true 表示全部子結點进行同样的操作

inheritChildren = false 表示不影响子結點

省略此参數，等同于 inheritChildren = false

返回值无

目前无任何返回值

function 举例

1. 禁用當前選中的結點的 checkbox / radio

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
for (var i=0, l=nodes.length; i < l; i++) {
	treeObj.setChkDisabled(nodes[i], true);
}
