Function(filter, isSingle, parentNode, invokeParam)zTreeObj.getNodesByFilter

概述[ 依赖 jquery.ztree.core 核心 js ]

根据自定义规则搜索結點資料 JSON 对象集合 或 單个結點資料

可自定义复杂的搜索规则

請通过 zTree 对象执行此方法。

Function 参數说明

filterFunction

自定义过滤器函數 function filter(node) {...}

filter 参數：node (結點資料 JSON)

filter 返回值：boolean (true 表示符合搜索条件；false 表示不符合搜索条件)

isSingleBoolean

isSingle = true 表示只查找單个結點

isSingle = false 表示查找結點集合

忽略此参數，表示查找結點集合

parentNodeJSON

可以指定在某个父結點下的子結點中搜索

忽略此参數，表示在全部結點中搜索

invokeParam任意类型

用户自定义的資料对象，用于 filter 中进行計算

返回值Array(JSON) / JSON

isSingle = true 返回 第一个找到的結點資料 JSON，无结果时返回 null

isSingle = false 返回 結點資料集合 Array(JSON)，无结果时返回 [ ]

function 举例

1. 查找 level = 2 & name 中包含 "test" 的結點資料

function filter(node) {
    return (node.level == 2 && node.name.indexOf("test")>-1);
}
......
var treeObj = $.fn.zTree.getZTreeObj("tree");
var node = treeObj.getNodesByFilter(filter, true); // 仅查找一个結點
var nodes = treeObj.getNodesByFilter(filter); // 查找結點集合
