Function(event, treeId, treeNode)setting.callback.onNodeCreated

概述[ 依赖 jquery.ztree.core 核心 js ]

用于捕獲結點生成 DOM 後的事件回调函數

v3.x 采用了延迟加載技术，因此对于父結點未展开的子結點来说，初始化後是不會触发此回调函數，直到其父結點被展开

大資料量的結點加載請注意：不设置 onNodeCreated，可以提升一部分初始化性能

默认值：null

Function 参數说明

eventjs event 对象

标准的 js event 对象

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

生成 DOM 完毕的結點的 JSON 資料对象

setting & function 举例

1. 创建結點 DOM 後， 弹出该結點的 tId、name 的信息

function zTreeOnNodeCreated(event, treeId, treeNode) {
    alert(treeNode.tId + ", " + treeNode.name);
};
var setting = {
	callback: {
		onNodeCreated: zTreeOnNodeCreated
	}
};
