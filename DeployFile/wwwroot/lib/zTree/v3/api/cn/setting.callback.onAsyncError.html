setting.callback.onAsyncError

Function(event, treeId, treeNode, XMLHttpRequest, textStatus, errorThrown) 

概述[ 依赖 jquery.ztree.core 核心 js ]

用于捕獲异步加載出现异常錯誤的事件回调函數

如果设置了 setting.callback.beforeAsync 方法，且返回 false，将无法触发 onAsyncSuccess / onAsyncError 事件回调函數。

默认值：null

Function 参數说明

eventjs event 对象

标准的 js event 对象

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

进行异步加載的父結點 JSON 資料对象

针对根进行异步加載时，treeNode = null

XMLHttpRequestString

标准 XMLHttpRequest 对象，請参考 JQuery API 文档。

textStatusString

請求状态：success，error，請参考 JQuery API 文档。

errorThrownString

errorThrown 只有当异常发生时才會被传递，請参考 JQuery API 文档。

setting & function 举例

1. 异步加載出现异常後，弹出錯誤信息

function zTreeOnAsyncError(event, treeId, treeNode, XMLHttpRequest, textStatus, errorThrown) {
    alert(XMLHttpRequest);
};
var setting = {
	callback: {
		onAsyncError: zTreeOnAsyncError
	}
};
