Function(event, treeId, treeNodes, targetNode, moveType, isCopy)setting.callback.onDrop

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用于捕獲結點拖拽操作结束的事件回调函數

如果设置了 setting.callback.beforeDrop 方法，且返回 false，将无法触发 onDrop 事件回调函數。

默认值：null

Function 参數说明

eventjs event 对象

标准的 js event 对象

treeIdString

目标結點 targetNode 所在 zTree 的 treeId，便于用户操控

treeNodesArray(JSON)

被拖拽的結點 JSON 資料集合

如果拖拽操作為 移动，treeNodes 是當前被拖拽結點的資料集合。

如果拖拽操作為 复制，treeNodes 是复制後 clone 得到的新結點資料。

targetNodeJSON

成為 treeNodes 拖拽结束的目标結點 JSON 資料对象。

如果拖拽成為根結點，则 targetNode = null

moveTypeString

指定移动到目标結點的相对位置

"inner"：成為子結點，"prev"：成為同级前一个結點，"next"：成為同级後一个結點

如果 moveType = null，表明拖拽无效

isCopyBoolean

拖拽結點操作是 复制 或 移动

true：复制；false：移动

setting & function 举例

1. 每次拖拽操作结束後， 弹出该被拖拽結點的个數以及目标結點的 tId、name 的信息

function zTreeOnDrop(event, treeId, treeNodes, targetNode, moveType) {
    alert(treeNodes.length + "," + (targetNode ? (targetNode.tId + ", " + targetNode.name) : "isRoot" ));
};
var setting = {
	callback: {
		onDrop: zTreeOnDrop
	}
};
