Boolean / Function(treeId, treeNode)setting.edit.showRemoveBtn

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

设置是否顯示删除按鈕。[setting.edit.enable = true 时生效]

当點擊某結點的删除按鈕时：

1、首先触发 setting.callback.beforeRemove 回调函數，用户可判定是否进行删除操作。

2、如果未设置 beforeRemove 或 beforeRemove 返回 true，则删除結點并触发 setting.callback.onRemove 回调函數。

默认值：true

Boolean 格式说明

true / false 分别表示 顯示 / 隱藏 删除按鈕

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

需要设置是否顯示删除按鈕的結點 JSON 資料对象

返回值Boolean

返回值同 Boolean 格式的資料

setting & function 举例

1. 不顯示删除按鈕

var setting = {
	edit: {
		enable: true,
		showRemoveBtn: false
	}
};
......

2. 设置所有的父結點不顯示删除按鈕

function setRemoveBtn(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		showRemoveBtn: setRemoveBtn
	}
};
......
