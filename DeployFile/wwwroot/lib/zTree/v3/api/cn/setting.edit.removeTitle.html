String / Function(treeId, treeNode)setting.edit.removeTitle

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

删除按鈕的 Title 辅助信息。[setting.edit.enable = true & setting.edit.showRemoveBtn = true 时生效]

默认值："remove"

String 格式说明

鼠标移动到 删除按鈕 上时，瀏覽器自动弹出的辅助信息內容，可根据用户需要自行修改

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

需要设置删除按鈕 Title 信息的結點 JSON 資料对象

返回值String

返回值同 String 格式的資料

setting & function 举例

1. 设置删除按鈕的 Title 辅助信息為: "删除結點"

var setting = {
	edit: {
		enable: true,
		showRemoveBtn: true,
		removeTitle: "删除結點"
	}
};
......

2. 设置父結點删除按鈕的 Title 辅助信息為: "删除父結點"

function setRemoveTitle(treeId, treeNode) {
	return treeNode.isParent ? "删除父結點":"删除叶子結點";
}
var setting = {
	edit: {
		enable: true,
		showRemoveBtn: true,
		removeTitle: setRemoveTitle
	}
};
......
