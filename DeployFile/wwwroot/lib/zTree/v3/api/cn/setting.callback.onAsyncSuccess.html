Function(event, treeId, treeNode, msg)setting.callback.onAsyncSuccess

概述[ 依赖 jquery.ztree.core 核心 js ]

用于捕獲异步加載正常结束的事件回调函數

如果设置了 setting.callback.beforeAsync 方法，且返回 false，将无法触发 onAsyncSuccess / onAsyncError 事件回调函數。

默认值：null

Function 参數说明

eventjs event 对象

标准的 js event 对象

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

进行异步加載的父結點 JSON 資料对象

针对根进行异步加載时，treeNode = null

msgString / Object

异步獲取的結點資料字符串，主要便于用户调试使用。

实际資料类型會受 setting.async.dataType 的设置影响，請参考 JQuery API 文档。

setting & function 举例

1. 异步加載成功後，弹出提示信息

function zTreeOnAsyncSuccess(event, treeId, treeNode, msg) {
    alert(msg);
};
var setting = {
	callback: {
		onAsyncSuccess: zTreeOnAsyncSuccess
	}
};
