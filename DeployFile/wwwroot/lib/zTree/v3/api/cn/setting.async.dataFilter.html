Function(treeId, parentNode, responseData)setting.async.dataFilter

概述[ 依赖 jquery.ztree.core 核心 js ]

用于对 Ajax 返回資料进行预處理的函數。[setting.async.enable = true 时生效]

默认值：null

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

parentNodeJSON

进行异步加載的父結點 JSON 資料对象

对根进行异步加載时，parentNode = null

responseDataArray(JSON) / JSON / String

异步加載獲取到的資料转换後的 Array(JSON) / JSON / String 資料对象

v3.4开始 支持 XML 資料格式的 String

返回值Array(JSON) / JSON

返回值是 zTree 支持的JSON 資料结构即可。

v3.x 支持單个 JSON 結點資料进行加載

setting & function 举例

1. 修改异步獲取到的結點name属性

function ajaxDataFilter(treeId, parentNode, responseData) {
    if (responseData) {
      for(var i =0; i < responseData.length; i++) {
        responseData[i].name += "_filter";
      }
    }
    return responseData;
};
var setting = {
	async: {
		enable: true,
		url: "http://host/getNode.php",
		dataFilter: ajaxDataFilter
	}
};
