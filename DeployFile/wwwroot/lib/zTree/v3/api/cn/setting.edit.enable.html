Booleansetting.edit.enable

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

设置 zTree 是否处于编辑状态

請在初始化之前设置，初始化後需要改变编辑状态請使用 zTreeObj.setEditable() 方法

默认值: false

Boolean 格式说明

true / false 分别表示 可以 / 不可以 编辑

编辑状态规则说明

1、點擊結點时，不會打开 node.url 指定的 URL。
2、全面支持 编辑 与 异步加載 状态共存。
3、可以对結點进行拖拽，且支持多棵树之间进行拖拽。
4、支持拖拽时 复制/移动 結點。（参考: setting.edit.drag.isCopy / setting.edit.drag.isMove）
5、可以通过编辑按鈕修改 name 属性。
6、可以通过删除按鈕删除結點。

請注意大小写，不要改变

setting 举例

1. 设置 zTree 进入编辑状态

var setting = {
	edit: {
		enable: true
	}
};
