Function(treeId, treeNode)setting.callback.beforeCollapse

概述[ 依赖 jquery.ztree.core 核心 js ]

用于捕獲父結點折叠之前的事件回调函數，并且根据返回值确定是否允許折叠操作

默认值：null

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

要折叠的父結點 JSON 資料对象

返回值Boolean

返回值是 true / false

如果返回 false，zTree 将不會折叠結點，也无法触发 onCollapse 事件回调函數

setting & function 举例

1. 禁止所有已展开的父結點折叠

function zTreeBeforeCollapse(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeCollapse: zTreeBeforeCollapse
	}
};
