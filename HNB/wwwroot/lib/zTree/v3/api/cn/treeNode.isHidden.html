BooleantreeNode.isHidden

概述[ 依赖 jquery.ztree.exhide 扩展 js ]

判断 treeNode 結點是否被隱藏。

1、初始化 zTree 时，如果結點设置 isHidden = true，會被自动隱藏

2、請勿对已加載的結點修改此属性，隱藏 / 顯示 請使用 hideNode() / hideNodes() / showNode() / showNodes() 方法

Boolean 格式说明

true 表示被隱藏

false 表示被顯示

treeNode 举例

1. 查看第一个根結點是否被隱藏

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getNodes();
if (sNodes.length > 0) {
	var isHidden = sNodes[0].isHidden;
}
