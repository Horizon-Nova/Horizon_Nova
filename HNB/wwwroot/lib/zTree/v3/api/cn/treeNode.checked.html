BooleantreeNode.checked

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

結點的 checkBox / radio 的 勾選状态。[setting.check.enable = true & treeNode.nocheck = false 时有效]

1、如果不使用 checked 属性设置勾選状态，請修改 setting.data.key.checked

2、建立 treeNode 資料时设置 treeNode.checked = true 可以让結點的输入框默认為勾選状态

3、修改結點勾選状态，可以使用 treeObj.checkNode / checkAllNodes / updateNode 方法，具体使用哪种請根据自己的需求而定

4、為了解决部分朋友生成 json 資料出现的兼容问题, 支持 "false","true" 字符串格式的資料

默认值：false

Boolean 格式说明

true 表示結點的输入框被勾選

false 表示結點的输入框未勾選

treeNode 举例

1. 初始化的資料设置 默认為勾選状态

var nodes = [
{ "id":1, "name":"test1", checked:true },
{ "id":2, "name":"test2", checked:true }
]

2. 獲取第一个根結點的勾選状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
var checked = treeObj.getNodes()[0].checked;
