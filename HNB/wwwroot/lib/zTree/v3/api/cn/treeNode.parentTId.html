StringtreeNode.parentTId

概述[ 依赖 jquery.ztree.core 核心 js ]

treeNode 結點的父結點唯一識別 tId。

1、v3.x 用 parentTId 替换了原先的 parentNode 属性，同时增加了 getParentNode 方法，以避免原先 parentNode 造成的 clone 死循环

2、初始化結點資料时，由 zTree 增加此属性，請勿提前赋值

String 格式说明

zTree 内部生成的結點唯一識別，請参考 treeNode.tId 的说明

如果 treeNode 是根結點，则 parentTId = null

treeNode 举例

1. 查看當前被選中的結點的父結點 tId

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var parentTId = sNodes[0].parentTId;
}
