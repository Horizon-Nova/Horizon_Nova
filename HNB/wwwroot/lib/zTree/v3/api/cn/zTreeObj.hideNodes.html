Function(treeNodes)zTreeObj.hideNodes

概述[ 依赖 jquery.ztree.exhide 扩展 js ]

隱藏一批結點。

1、此功能不支持 exedit 扩展，因此不要在编辑状态时使用隱藏結點的方法。

2、隱藏/顯示結點，會影响結點的 isFirstNode 和 isLastNode 属性。

3、請通过 zTree 对象执行此方法。

Function 参數说明

treeNodesArray(JSON)

指定被隱藏的結點 JSON 資料集合

請务必保证这些結點資料对象 是 zTree 内部的資料对象

返回值无

目前无任何返回值

function 举例

1. 隱藏根結點第一个結點的子結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.hideNodes(nodes[0].children);
