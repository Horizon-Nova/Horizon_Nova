Booleansetting.view.selectedMulti

概述[ 依赖 jquery.ztree.core 核心 js ]

设置是否允許同时選中多个結點。

默认值: true

Boolean 格式说明

true / false 分别表示 支持 / 不支持 同时選中多个結點

1、设置為 true时，按下 Ctrl 或 Cmd 键可以選中多个結點

2、设置為 true / false 都不影响按下 Ctrl 或 Cmd 键可以让已選中的結點取消選中状态（ 取消選中状态可以参考 setting.view.autoCancelSelected ）

setting 举例

1. 禁止多點同时選中的功能

var setting = {
	view: {
		selectedMulti: false
	}
};
