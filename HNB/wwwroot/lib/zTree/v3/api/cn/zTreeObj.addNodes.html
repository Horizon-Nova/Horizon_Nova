Function(parentNode, newNodes, isSilent)zTreeObj.addNodes

概述[ 依赖 jquery.ztree.core 扩展 js ]

新增結點。

v3.x 為了避免原先反复初始化造成的資料重复问题，在 初始化 和 新增結點 时内部进行 clone 操作。如果需要獲取資料在 zTree 内的对象，請獲取此方法的返回值。

請通过 zTree 对象执行此方法。

Function 参數说明

parentNodeJSON

指定的父結點，如果增加根結點，請设置 parentNode 為 null 即可。

請务必保证此結點資料对象 是 zTree 内部的資料对象

newNodesJSON / Array(JSON)

需要增加的結點資料 JSON 对象集合，資料只需要满足 zTree 的結點資料必需的属性即可，详细請参考“treeNode 結點資料详解”

1、v3.x 支持單独新增一个結點，即如果只新增一个結點，不用必须包在數组中

2、使用简單資料模式，請参考 setting.data.simpleData 内的属性说明

isSilentBoolean

设定增加結點後是否自动展开父結點。

isSilent = true 时，不展开父結點，其他值或缺省状态都自动展开。

返回值Array(JSON)

返回值是 zTree 最终新增的結點資料集合

如果 newNodes 是單个結點資料 JSON，返回值也是将其包在 Array 内

請务必记住：返回值中的資料对象 是 newNodes 被 clone 後的，所以绝对不相等！

function 举例

1. 对于 id = "tree" 的 zTree 增加 1 个根結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var newNode = {name:"newNode1"};
newNode = treeObj.addNodes(null, newNode);

2. 对于 id = "tree" 的 zTree 增加 3 个根結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var newNodes = [{name:"newNode1"}, {name:"newNode2"}, {name:"newNode3"}];
newNodes = treeObj.addNodes(null, newNodes);
