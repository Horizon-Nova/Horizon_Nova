Function(targetNode, treeNode, moveType, isSilent)zTreeObj.copyNode

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

复制結點。

v3.x 复制結點时进行 clone 操作。如果需要獲取資料在 zTree 内的对象，請獲取此方法的返回值。

請通过 zTree 对象执行此方法。

Function 参數说明

targetNodeJSON

要复制到的目标結點 JSON 資料

如果复制成為根結點，請设置 targetNode 為 null 即可

請务必保证此結點資料对象 是 zTree 内部的資料对象

treeNodeJSON

需要被复制的結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

moveTypeString

复制到目标結點的相对位置

"inner"：成為子結點，"prev"：成為同级前一个結點，"next"：成為同级後一个結點

isSilentBoolean

设定复制結點後是否自动展开父結點。

isSilent = true 时，不展开父結點，其他值或缺省状态都自动展开。

返回值JSON

返回值是最终加入到 zTree 内的結點資料

請务必记住：返回值中的資料对象 是 treeNode 被 clone 後的，所以绝对不相等！

function 举例

1. 将根結點中第二个結點 复制成為 第一个結點的子結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.copyNode(nodes[0], nodes[1], "inner");

2. 将根結點中第二个結點 复制成為 第一个結點的前一个結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
treeObj.copyNode(nodes[0], nodes[1], "before");
