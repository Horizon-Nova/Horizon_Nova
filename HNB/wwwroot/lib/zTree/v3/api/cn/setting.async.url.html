String / Function(treeId, treeNode)setting.async.url

概述[ 依赖 jquery.ztree.core 核心 js ]

Ajax 獲取資料的 URL 地址。[setting.async.enable = true 时生效]

默认值：""

String 格式说明

设置固定的异步加載 url 字符串，請注意地址的路径，确保頁面能正常加載

url 内也可以带参數，这些参數就只能是通过 get 方式提交了，并且請注意进行转码

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

需要异步加載子結點的的父結點 JSON 資料对象

针对根进行异步加載时，treeNode = null

返回值String

返回值同 String 格式的資料

setting & function 举例

1. 设置异步獲取結點的 URL 為 nodes.php

var setting = {
	async: {
		enable: true,
		url: "nodes.php",
		autoParam: ["id", "name"]
	}
};
......

2. 设置异步獲取結點的 URL 為 function 动态獲取

function getAsyncUrl(treeId, treeNode) {
    return treeNode.isParent ? "nodes1.php" : "nodes2.php";
};
var setting = {
	async: {
		enable: true,
		url: getAsyncUrl,
		autoParam: ["id", "name"]
	}
};
......
