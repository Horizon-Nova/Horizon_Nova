BooleantreeNode.checkedOld

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

結點的 checkBox / radio 在初始化时的 勾選状态。[setting.check.enable = true & treeNode.nocheck = false 时有效]

1、zTree 初始化結點資料时會对此属性进行赋值，因此請勿对此属性初始化

2、如需配合 zTreeObj.getChangeCheckedNodes 方法实现特殊功能，可以根据需求在使用中自行修改 checkedOld 資料

默认值：checked的初始化值

Boolean 格式说明

true 表示結點初始化时 输入框被勾選

false 表示結點初始化时 输入框未勾選

treeNode 举例

1. 獲取第一个根結點的初始勾選状态

var treeObj = $.fn.zTree.getZTreeObj("tree");
var checkedOld = treeObj.getNodes()[0].checkedOld;
