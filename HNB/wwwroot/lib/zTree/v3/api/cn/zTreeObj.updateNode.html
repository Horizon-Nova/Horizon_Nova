Function(treeNode, checkTypeFlag)zTreeObj.updateNode

概述[ 依赖 jquery.ztree.core 核心 js ]

更新某結點資料，主要用于该結點顯示属性的更新。

1、可针对 name、target、 url、icon、 iconSkin、checked、nocheck 等这几个用于顯示效果的参數进行更新，其他用于 zTreeNodes 的参數請不要随意更新，对于展开結點，还請调用 expandNode方法，因此請勿随意修改 open 属性。

2、用此方法修改 checked 勾選状态不會触发 beforeCheck / onCheck 事件回调函數。

請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

指定需要更新的結點 JSON 資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

checkTypeFlagBoolean

checkTypeFlag = true 表示按照 setting.check.chkboxType 属性进行父子結點的勾選联动操作

checkTypeFlag = false 表示只修改此結點勾選状态，无任何勾選联动操作

当 setting.check.enable = true 且 setting.check.chkStyle = "checkbox" 时才有效

不影响父子結點中 treeNode.nochecked = true 的結點。

返回值无

目前无任何返回值

function 举例

1. 更新根結點中第一个結點的名称

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
if (nodes.length>0) {
	nodes[0].name = "test";
	treeObj.updateNode(nodes[0]);
}
