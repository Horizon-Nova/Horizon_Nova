Function(treeNode)zTreeObj.getNodeIndex

概述[ 依赖 jquery.ztree.core 核心 js ]

獲取某結點在同级結點中的序号（從0开始）

請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

需要查询顺序的結點 JSON 資料对象

請务必保证此結點資料对象 是 zTree 内部的資料对象

返回值Number

返回值從 0 开始计數

如果不存在该結點資料，返回 -1

function 举例

1. 獲取當前選中的第一个結點在同级結點中的序号

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
if (nodes.length>0) {
	var index = treeObj.getNodeIndex(nodes[0]);
}
