BooleantreeNode.zAsync

概述[ 依赖 jquery.ztree.core 核心 js ]

记录 treeNode 結點是否已经进行过异步加載，避免父結點反复异步加載資料。

初始化結點資料时，由 zTree 增加此属性，請勿提前赋值

默认值：false （无子結點的父結點）； true （有子結點的父結點 & 叶子結點）

Boolean 格式说明

true 表示父結點展开时不需要自动异步加載

false 表示父結點展开时需要自动异步加載

此参數不會对 reAsyncChildNodes 方法造成任何影响

treeNode 举例

1. 查看當前被選中的結點是否結點是否需要自动异步加載

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var zAsync = sNodes[0].zAsync;
}
