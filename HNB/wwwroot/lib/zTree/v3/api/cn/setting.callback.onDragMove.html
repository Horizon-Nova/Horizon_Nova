Function(event, treeId, treeNodes)setting.callback.onDragMove

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

用于捕獲結點被拖拽过程中移动的事件回调函數

主要用于捕獲 zTree 結點拖拽到的 DOM，從而操作对应的 DOM。

默认值：null

Function 参數说明

eventjs event 对象

标准的 js event 对象

treeIdString

被拖拽的結點 treeNodes 所在 zTree 的 treeId，便于用户操控

treeNodesArray(JSON)

要被拖拽的結點 JSON 資料集合

setting & function 举例

1. 拖拽結點时，随时输出 當前拖拽到的目标 DOM

function zTreeOnDragMove(event, treeId, treeNodes) {
    console.log(event.target);
};
var setting = {
	callback: {
		onDragMove: zTreeOnDragMove
	}
};
