StringtreeNode.icon

概述[ 依赖 jquery.ztree.core 核心 js ]

結點自定义圖标的 URL 路径。

1、父結點如果只设置 icon ，會導致展开、折叠时都使用同一个圖标

2、父結點展开、折叠使用不同的个性化圖标需要同时设置 treeNode.iconOpen / treeNode.iconClose 两个属性

3、如果想利用 className 设置个性化圖标，需要设置 treeNode.iconSkin 属性

默认值：无

String 格式说明

圖标圖片的 url 可以是相对路径也可以是绝对路径

设置相对路径請注意頁面与圖片之间的关系，确保圖片能够正常加載

treeNode 举例

1. 设置結點的个性化圖标

var nodes = [
	//父結點展开 折叠时使用相同的圖标
	{ name:"父結點1", icon:"/img/parent.gif"},

	//父結點展开 折叠时分别使用不同的圖标
	{ name:"父結點2", iconOpen:"/img/open.gif", iconClose:"/img/close.gif"}, 

	//叶子結點个性化圖标
	{ name:"叶子結點", icon:"/img/leaf.gif"}
]
