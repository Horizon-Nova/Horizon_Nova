Function(treeId, treeNode)setting.callback.beforeExpand

概述[ 依赖 jquery.ztree.core 核心 js ]

用于捕獲父結點展开之前的事件回调函數，并且根据返回值确定是否允許展开操作

默认值：null

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

要展开的父結點 JSON 資料对象

返回值Boolean

返回值是 true / false

如果返回 false，zTree 将不會展开結點，也无法触发 onExpand 事件回调函數

setting & function 举例

1. 禁止所有已折叠的父結點展开

function zTreeBeforeExpand(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeExpand: zTreeBeforeExpand
	}
};
