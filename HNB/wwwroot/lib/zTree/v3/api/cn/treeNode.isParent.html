BooleantreeNode.isParent

概述[ 依赖 jquery.ztree.core 核心 js ]

记录 treeNode 結點是否為父結點。

1、初始化結點資料时，根据 treeNode.children 属性判断，有子結點则设置為 true，否则為 false

2、初始化結點資料时，如果设定 treeNode.isParent = true，即使无子結點資料，也會设置為父結點

3、為了解决部分朋友生成 json 資料出现的兼容问题, 支持 "false","true" 字符串格式的資料

Boolean 格式说明

true 表示是父結點

false 表示不是父結點

treeNode 举例

1. 查看當前被選中的結點是否是父結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var sNodes = treeObj.getSelectedNodes();
if (sNodes.length > 0) {
	var isParent = sNodes[0].isParent;
}
