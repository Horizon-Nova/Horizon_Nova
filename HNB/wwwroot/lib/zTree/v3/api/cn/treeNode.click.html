StringtreeNode.click

概述[ 依赖 jquery.ztree.core 核心 js ]

最简單的 click 事件操作。相当于 onclick="..." 的內容。 如果操作较复杂，請使用 onClick 事件回调函數。

由于 IE 对于 onclick 和 click事件共存时的處理与其他瀏覽器不同，所以請不要利用此参數控制是否允許跳转的操作（例如：treeNode.click = "return false;"）。如有类似需求，請不要使用 url 属性设置网址，同时利用 onClick 回调函數控制跳转。

默认值：无

String 格式说明

标准 javascript 語法， 例如：alert("test"); 等

treeNode 举例

1. 设置某結點點擊时，弹出信息框

var nodes = [
	{ "id":1, "name":"Google CN", "url":"http://g.cn", "click":"alert('test');"},
	......
]
