Function(simpleNodes)zTreeObj.transformTozTreeNodes

概述[ 依赖 jquery.ztree.core 核心 js ]

将简單 Array 格式資料转换為 zTree 使用的标准 JSON 嵌套資料格式。

使用此方法，請务必设置結點唯一識別属性名称 setting.data.simpleData.idKey 和 父結點唯一識別属性名称 setting.data.simpleData.pIdKey，并且让資料满足父子关系。

請通过 zTree 对象执行此方法。

Function 参數说明

simpleNodesArray(JSON) / JSON

需要被转换的简單 Array 格式資料 或 某个單独的資料对象

返回值Array(JSON)

zTree 使用的标准資料，子結點都存在于父結點資料的 children 属性中

如果 simpleNodes 是一个 JSON 对象，则被简單封装成长度為 1 的數组。

function 举例

1. 将简單 Array 格式转换為zTree使用的标准格式

var setting = {
	data: {
		simpleData: {
			enable: true,
			idKey: "id",
			pIdKey: "pId",
			rootPId: 0
		}
	}
};
var simpleNodes = [
    {"id":1, "pId":0, "name":"test1"},
    {"id":11, "pId":1, "name":"test11"},
    {"id":12, "pId":1, "name":"test12"},
    {"id":111, "pId":11, "name":"test111"}
];
var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.transformTozTreeNodes(simpleNodes);
