Function(treeNode, addFlag)zTreeObj.selectNode

概述[ 依赖 jquery.ztree.core 核心 js ]

選中指定結點

v3.x 支持同时選中多个結點。

請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

需要被選中的結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

addFlagBoolean

addFlag = true 表示追加選中，會出现多點同时被選中的情况

addFlag = false 表示單独選中，原先被選中的結點會被取消選中状态

setting.view.selectedMulti = false 时，此参數无效，始终进行單独選中

返回值无

目前无任何返回值

function 举例

1. 單独選中根結點中第一个結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
if (nodes.length>0) {
	treeObj.selectNode(nodes[0]);
}
