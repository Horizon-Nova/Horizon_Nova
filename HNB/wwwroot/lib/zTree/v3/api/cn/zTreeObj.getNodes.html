Function()zTreeObj.getNodes

概述[ 依赖 jquery.ztree.core 核心 js ]

獲取 zTree 的全部結點資料

請通过 zTree 对象执行此方法。

Function 参數说明

返回值Array(JSON)

全部結點資料

1、Array 仅仅是根結點的集合（默认情况子結點都处于 children 属性下）；

2、如需遍历全部結點需要利用递归，或利用 transformToArray 方法 将資料变成简單的 Array 集合

3、对于异步加載模式下，尚未加載的子結點是无法通过此方法獲取的。

function 举例

1. 獲取全部結點資料

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getNodes();
