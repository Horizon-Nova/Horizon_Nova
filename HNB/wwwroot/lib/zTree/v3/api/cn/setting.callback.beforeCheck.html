Function(treeId, treeNode)setting.callback.beforeCheck

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

用于捕獲 勾選 或 取消勾選 之前的事件回调函數，并且根据返回值确定是否允許 勾選 或 取消勾選

默认值：null

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

进行 勾選 或 取消勾選 的結點 JSON 資料对象

返回值Boolean

返回值是 true / false

如果返回 false，将不會改变勾選状态，并且无法触发 onCheck 事件回调函數

setting & function 举例

1. 禁止所有勾選操作，保持初始化的勾選状态

function zTreeBeforeCheck(treeId, treeNode) {
    return false;
};
var setting = {
	callback: {
		beforeCheck: zTreeBeforeCheck
	}
};
