Function(treeNode, callbackFlag)zTreeObj.removeNode

概述[ 依赖 jquery.ztree.core 扩展 js ]

删除結點。

v3.x 中删除結點可以触发 beforeRemove / onRemove 事件回调函數。便于减少冗余代码

請通过 zTree 对象执行此方法。

Function 参數说明

treeNodeJSON

需要被删除的結點資料

請务必保证此結點資料对象 是 zTree 内部的資料对象

callbackFlagBoolean

callbackFlag = true 表示执行此方法时触发 beforeRemove & onRemove 事件回调函數

callbackFlag = false 表示执行此方法时不触发事件回调函數

省略此参數，等同于 false

返回值无

目前无任何返回值

function 举例

1. 删除所有選中的結點

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getSelectedNodes();
for (var i=0, l=nodes.length; i < l; i++) {
	treeObj.removeNode(nodes[i]);
}
