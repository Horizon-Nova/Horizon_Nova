Boolean / Function(treeId, treeNode)setting.edit.showRenameBtn

概述[ 依赖 jquery.ztree.exedit 扩展 js ]

设置是否顯示编辑名称按鈕。[setting.edit.enable = true 时生效]

当點擊某結點的编辑名称按鈕时：

1、进入結點编辑名称状态。

2、编辑名称完毕（Input 失去焦點 或 按下 Enter 键），會触发 setting.callback.beforeRename 回调函數，用户可根据自己的规则判定是否允許修改名称。

3、如果 beforeRename 返回 false，则继续保持编辑名称状态，直到名称符合规则位置 （按下 ESC 键可取消编辑名称状态，恢复原名称）。

4、如果未设置 beforeRename 或 beforeRename 返回 true，则结束結點编辑名称状态，更新結點名称，并触发 setting.callback.onRename 回调函數。

默认值：true

Boolean 格式说明

true / false 分别表示 顯示 / 隱藏 编辑名称按鈕

Function 参數说明

treeIdString

对应 zTree 的 treeId，便于用户操控

treeNodeJSON

需要设置是否顯示编辑名称按鈕的結點 JSON 資料对象

返回值Boolean

返回值同 Boolean 格式的資料

setting & function 举例

1. 不顯示编辑名称按鈕

var setting = {
	edit: {
		enable: true,
		showRenameBtn: false
	}
};
......

2. 设置所有的父結點不顯示编辑名称按鈕

function setRenameBtn(treeId, treeNode) {
	return !treeNode.isParent;
}
var setting = {
	edit: {
		enable: true,
		showRenameBtn: setRenameBtn
	}
};
......
