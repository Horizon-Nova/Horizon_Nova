Function(checked)zTreeObj.getCheckedNodes

概述[ 依赖 jquery.ztree.excheck 扩展 js ]

獲取输入框被勾選 或 未勾選的結點集合。[setting.check.enable = true 时有效]

請通过 zTree 对象执行此方法。

Function 参數说明

checkedBoolean

checked = true 表示獲取 被勾選 的結點集合

checked = false 表示獲取 未勾選 的結點集合

省略此参數，等同于 true。

对于 treeNode.nochecked = true 的結點不进行獲取。

返回值Array(JSON)

返回全部符合要求的結點集合 Array

function 举例

1. 獲取當前被勾選的結點集合

var treeObj = $.fn.zTree.getZTreeObj("tree");
var nodes = treeObj.getCheckedNodes(true);
